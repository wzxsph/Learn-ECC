# 入门练习

完成以下练习，巩固 ECC 基础。

## 练习 1：环境验证

**目标**：确认 ECC 环境正确安装

**步骤**：
1. 运行测试套件：`node tests/run-all.js`
2. 验证所有测试通过
3. 列出 `.claude/` 目录结构

**验收标准**：
- 测试全部通过
- 能看到 agents、skills、hooks、rules 目录

---

## 练习 2：使用斜杠命令

**目标**：熟悉常用斜杠命令

**步骤**：
1. 启动 Claude Code
2. 依次执行以下命令，观察输出：
   - `/help` 或 `/ecc:plan`
   - `/ecc:code-review`
3. 记录每个命令的输出格式

**验收标准**：
- 了解各命令的功能和输出格式

---

## 练习 3：创建简单 Agent

**目标**：了解 Agent 文件格式

**步骤**：
1. 查看 `agents/` 目录下的现有 Agent 文件
2. 参考 `planner.md` 的格式
3. 创建一个简单的 `hello-agent.md`，功能为输出问候语

**验收标准**：
- 新 Agent 文件格式正确
- 包含 YAML frontmatter（name, description, tools, model）

---

## 练习 4：运行 Hook

**目标**：了解 Hook 机制

**步骤**：
1. 查看 `hooks/` 目录下的 Hook 配置
2. 运行 `node scripts/hooks/run-with-flags.js --help`（如果支持）
3. 理解 Hook 的触发时机和执行逻辑

**验收标准**：
- 能描述 Hook 的工作原理
- 了解 PreToolUse 和 PostToolUse 的区别

---

## 练习 5：综合任务

**目标**：完成一个小项目

**任务**：创建一个人名格式化工具

**功能要求**：
- `formatName(name)` - 格式化为「姓, 名」格式
- `initials(name)` - 返回首字母缩写

**执行流程**：
1. 使用 `/ecc:plan` 规划
2. 使用 `/tdd` 开发
3. 使用 `/ecc:code-review` 审查

**验收标准**：
- 完整实现两个函数
- 包含测试用例
- 通过代码审查
