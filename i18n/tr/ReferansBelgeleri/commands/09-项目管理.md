# Proje Yönetimi命令

## Genel Bakış

Proje Yönetimi命令Kullanılan项目配置、基础设施设置和工具链Yönet。

## 命令列表

### /projects

**Kullanım**: 列出已知项目 + instinct 统计

**Tanım**: 列出用户所有已知项目及其相关的 instinct 统计信息。

---

### /harness-audit

**Kullanım**: 审计 agent harness 配置

**Tanım**: 检查和审计 Claude Code 的 harness 配置。

**检查内容**:
- 工具链配置
- 代理设置
- MCP Sunucusu状态
- 规则配置

---

### /model-route

**Kullanım**: 路由Görev到合适模型

**Tanım**: 根据GörevTür选择最合适的模型（Haiku/Sonnet/Opus）。

**模型选择指南**:
- **Haiku**: 轻量级Görev、代码生成、频繁调用
- **Sonnet**: 主要开发工作、复杂编码Görev
- **Opus**: 架构决策、深度推理、研究分析

---

### /pm2

**Kullanım**: PM2 进程Yönet器初始化

**Tanım**: 初始化 PM2 进程Yönet器配置。

---

### /setup-pm

**Kullanım**: 配置包Yönet器

**Tanım**: 配置项目使用的包Yönet器。

**Destekle的包Yönet器**:
- npm
- pnpm
- yarn
- bun

**可Vasıtasıyla** `CLAUDE_PACKAGE_MANAGER` **环境变量配置**

---

### /project-init

**Kullanım**: 项目初始化

**Tanım**: 初始化新项目的标准结构和配置。

---

## 相关命令

- `/projects` - 列出项目
- `/harness-audit` - 审计配置
- `/model-route` - 模型路由
