## code-reviewer

### Ad和Kullanım
通用Kod İnceleme专家，主动Kodu İncele的质量、Güvenlik性和可维护性。所有代码变更后Zorunlu使用。

### 能力Açıklama
- Kod Kalitesi检查（函数大小、嵌套深度、代码重复）
- Güvenlik Açıkları检测（SQL Enjeksiyonu、XSS、Sabit Kodlu Kimlik Bilgileri）
- React/Next.js 模式检查
- Node.js/后端模式检查
- Performans Sorunu Tespiti
- En İyi Uygulamalar建议

### Uygulama Senaryosu
- 代码写入或修改后
- Pull Request 审查
- 合并到共享分支前

### Kullanılan Araçlar Listesi
- Read: 读取文件内容
- Grep: 搜索代码模式
- Glob: 查找文件
- Bash: 运行 git diff 和 lint 命令

### 与其他 Ajan 的协作方式
- 发现Güvenlik问题时 Escalation 到 security-reviewer
- 发现构建错误时使用 build-error-resolver
- 发现Gerekli重构时使用 refactor-cleaner
