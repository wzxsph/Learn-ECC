# Backend ve API Becerileri

## Genel Bakış

Backend ve API ile ilgili beceriler, sunucu tarafı uygulamaları, API Tasarımı ve veritabanı işlemlerini oluşturmak için Kullanılan.

## Çekirdek Beceriler

### backend-patterns

**Kullanım**: Backend geliştirme En İyi Uygulamaları

**Temel Kavramlar**:
- Katmanlı mimari (Controller → Service → Repository)
- Bağımlılık enjeksiyonu
- Hata Yönetimi stratejileri
- Önbellek kalıpları
- Mesaj kuyruğu entegrasyonu

---

### api-design

**Kullanım**: API Tasarımı ve RESTful standartları

**Temel Kavramlar**:
- RESTful rota tasarımı
- İstek/yanıt formatı
- Kimlik doğrulama ve yetkilendirme
- Sürüm kontrolü stratejileri
- Hız sınırlama ve önbellekleme

---

### api-connector-builder

**Kullanım**: API bağlayıcı oluşturma araçları

**Kullanım Senaryosu**:
- Üçüncü taraf API entegrasyonu
- Webhook işleme
- OAuth süreçlerini Gerçekleştir
- API hata yeniden deneme mekanizmaları

---

## Veritabanı Becerileri

### database-migrations

**Kullanım**: Veritabanı geçişlerini yönetme

**Temel Kavramlar**:
- Geçiş dosyası organizasyonu
- Veri geri alma stratejileri
- Tohum verisi yönetimi
- Sıfır kesinti süreli yayınlama

---

### mysql-patterns

**Kullanım**: MySQL veritabanı En İyi Uygulamaları

**Temel Kavramlar**:
- İndeks optimizasyonu
- Sorgu optimizasyonu
- Tablo bölme stratejileri
- Ana-aktif çoğaltma

---

### postgres-patterns

**Kullanım**: PostgreSQL veritabanı En İyi Uygulamaları

**Temel Kavramlar**:
- JSONB işlemleri
- Tam metin arama
- Bölümlendirilmiş tablolar
- İleri düzey indeks stratejileri

---

### redis-patterns

**Kullanım**: Redis önbellek kalıpları

**Temel Kavramlar**:
- Önbellek stratejileri (Cache-Aside, Write-Through)
- Dağıtık kilitler
- Oturum depolama
- Mesaj yayınlama/abone olma

---

### clickhouse-io

**Kullanım**: ClickHouse sütunlu depolama

**Kullanım Senaryosu**:
- Analitik sorgular
- Günlük depolama
- Gerçek zamanlı analiz
- Büyük veri boru hatları

---

## Mesaj Kuyrukları

### logistics-exception-management

**Kullanım**: Lojistik istisna yönetimi

**Kullanım Senaryosu**:
- Sipariş istisna işleme
- İade süreçleri
- Envanter senkronizasyonu

---
