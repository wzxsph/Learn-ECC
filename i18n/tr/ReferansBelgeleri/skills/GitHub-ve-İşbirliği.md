# GitHub 与协作技能

## Genel Bakış

GitHub 与协作技能Kullanılan GitHub 操作、Kod İnceleme、PR Yönet和团队协作。

## 核心技能

### github-ops

**Kullanım**: GitHub 操作与自动化

**功能**:
- Issue Yönet
- PR 创建与审查
- Kod İnceleme自动化
- GitHub Actions 工作流

---

### git-workflow

**Kullanım**: Git 工作流规范

**Temel Kavramlar**:
- 分支策略
- Commit 规范
- Merge 策略
- 冲突解决

---

### repo-scan

**Kullanım**: 代码库扫描

**检查项**:
- 依赖漏洞
- 代码重复
- 圈复杂度
- 文档完整性

---

## Kod İnceleme

### click-path-audit

**Kullanım**: 点击路径审计

**Kullanım Senaryosu**:
- 用户行为分析
- 转化率优化
- 漏斗分析

---

## 相关技能

- `github-ops` - GitHub 操作
- `git-workflow` - Git 工作流
- `repo-scan` - 代码库扫描
- `click-path-audit` - 点击路径审计
