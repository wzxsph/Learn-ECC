# 后端与 API 技能

## Genel Bakış

后端与 API 相关技能Kullanılan构建服务器端应用、API Tasarım和数据库操作。

## 核心技能

### backend-patterns

**Kullanım**: 后端开发En İyi Uygulamalar

**Temel Kavramlar**:
- 分层架构（Controller → Service → Repository）
- 依赖注入
- Hata Yönetimi策略
- 缓存模式
- 消息队列集成

---

### api-design

**Kullanım**: API Tasarım与RESTful 规范

**Temel Kavramlar**:
- RESTful 路由Tasarım
- 请求/响应格式
- 认证与授权
- 版本控制策略
- 限流与缓存

---

### api-connector-builder

**Kullanım**: API 连接器构建工具

**Kullanım Senaryosu**:
- 第三方 API 集成
- Webhook 处理
- OAuth 流程Gerçekleştir
- API 错误重试机制

---
