# Ajans İşbirliğiAlıştırma

## AlıştırmaHedef

学会在 Claude Code 中协调多个 Ajan Bitir复杂Görev。

## Alıştırma内容

### Alıştırma 1：Paralel Görev Yürütme
启动 2 个Paralel Ajan，分别Bitir：
- Ajan A：Proje Yapısını Analiz Et
- Ajan B：Kod Standartlarını Kontrol Et

### Alıştırma 2：Zincirleme Görev
Gerçekleştir 3 个 Ajan 的Zincirleme Çağrı：
- Ajan 1：Gereksinimleri Topla
- Ajan 2：Kod Üret
- Ajan 3：Kodu İncele

### Alıştırma 3：Sonuç Birleştirme
将多个 Ajan 的输出合并为统一报告。

## Şablon

```markdown
# Ajans İşbirliği方案

## GörevTanım
简述要Bitir的协作Görev。

## Katılımcı Ajanlar
| Ajan | Rol | Görev |
|-------|------|------|
| agent-a | 分析 | xxx |
| agent-b | 执行 | xxx |

## Yürütme Akışı
1. 启动 Ajan A
2. 启动 Ajan B
3. Sonuçları Birleştir

## Beklenen Çıktı
TanımSon Çıktı Formatı。
```

## Doğrulama
检查各 Ajan 输出是否正确聚合。
