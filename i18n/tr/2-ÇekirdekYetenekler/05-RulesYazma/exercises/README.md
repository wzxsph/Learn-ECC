# Rules YazımıAlıştırma

## AlıştırmaHedef

掌握 Claude Code 规则的创建和组织。

## Alıştırma内容

### Alıştırma 1：创建基础规则
创建 `my-rule.md`，包含：
- 规则目的Açıklama
- Uygulama Senaryosu定义
- 具体规范条款

### Alıştırma 2：Gerçekleştir多级规则
创建分层规则：
- 全局通用规则
- 项目特定规则
- 语言相关规则

### Alıştırma 3：规则调试
调试规则优先级和覆盖逻辑。

## Rule Şablon

```markdown
# 我的规则

## 目的
Tanım规则要解决什么问题。

## 适用范围
Tanım规则适用的Senaryo或文件Tür。

## 规范
- 规范 1
- 规范 2

## Doğrulama方式
Tanım如何Doğrulama规则是否被遵守。
```

## Doğrulama
1. 触发规则Senaryo
2. 检查规则是否生效
3. Doğrulama优先级覆盖
