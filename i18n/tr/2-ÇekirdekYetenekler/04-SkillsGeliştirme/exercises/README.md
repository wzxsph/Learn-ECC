# Skills GeliştirmeAlıştırma

## AlıştırmaHedef

学会创建和调试 Claude Code Skill 工作流。

## Alıştırma内容

### Alıştırma 1：创建基础 Skill
创建 `my-skill.md`，包含：
- When to Use 部分
- How It Works 部分
- 至少 2 个使用Örnek

### Alıştırma 2：Gerçekleştir Skill 工作流
创建完整 Skill：
- 定义触发条件
- 编写执行Adımlar
- 添加Hata Yönetimi

### Alıştırma 3：Skill 调试
调试现有 Skill，添加日志和Doğrulama逻辑。

## Skill Şablon

```markdown
# 我的 Skill

## When to Use
Tanım何时使用此 Skill。

## How It Works
Tanım Skill 的工作原理和Yürütme Akışı。

## Examples

### Örnek 1
Tanım第一个Kullanım Senaryosu。

### Örnek 2
Tanım第二个Kullanım Senaryosu。

## Not事项
- Not点 1
- Not点 2
```

## Doğrulama
1. 调用 Skill Doğrulama功能
2. 检查输出是否符合预期
