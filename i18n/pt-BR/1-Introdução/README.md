# Fase 1: Iniciante

Esta fase é projetada para usuários sem base prévia e ajuda você a entender e usar ECC rapidamente.

## Objetivos de Aprendizado

Após completar esta fase, você será capaz de:
- Entender os conceitos básicos e casos de uso do ECC
- Concluir a instalação e configuração básica do Claude Code
- Usar comandos frequentes com fluência
- Completar tarefas de desenvolvimento simples de forma independente

## Duração Estimada

2-3 horas

## Pré-requisitos

- Noções básicas de linha de comando
- Noções básicas de JavaScript/Node.js (apenas compreensão)
- Conceitos básicos de Git
