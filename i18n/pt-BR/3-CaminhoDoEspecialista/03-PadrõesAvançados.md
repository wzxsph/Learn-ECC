# 03-高级模式

掌握 ECC 的高级设计模式和最佳实践。

## 模式分类

> 参考: [最佳实践](../DocumentosDeReferência/skills/最佳实践.md)

### 1. 工作流模式
- **管道模式** - 数据流经多个处理阶段
- **广播模式** - 一个任务分发到多个执行者
- **回压模式** - 控制任务生产消费速率

### 2. Agent 模式
- **层级模式** - 主 Agent 协调子 Agents
- **市场模式** - Agents 通过竞价获取任务
- **蜂巢模式** - 大量同类 Agents 协作

### 3. 错误处理模式
- **熔断器模式** - 失败后快速失败
- **重试模式** - 指数退避重试
- **死信队列** - 无法处理的任务暂存

## 性能优化

### 上下文管理
- 及时清理无用上下文
- 使用压缩减少 token 消耗
- 分离冷热数据

### 并发控制
- 限制并发 Agent 数量
- 使用信号量控制资源
- 实现优雅降级

## 安全性

### 权限控制
- 最小权限原则
- 操作审计日志
- 敏感操作二次确认

### 输入验证
- 所有用户输入验证
- 参数化查询防注入
- 输出内容脱敏
