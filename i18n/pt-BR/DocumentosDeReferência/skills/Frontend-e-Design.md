# Frontend-e-Design技能

## 概述

Frontend-e-Design技能用于构建现代 Web 应用、UI/UX 设计和前端架构。

## 核心技能

### frontend-patterns

**用途**: 前端开发Melhores-Práticas

**核心概念**:
- 组件架构
- 状态管理
- Otimização-de-Desempenho
- 响应式设计
- 无障碍访问

---

### frontend-design-direction

**用途**: 前端设计方向指导

**涵盖内容**:
- 设计系统
- 设计令牌
- 主题化
- 品牌一致性

---

### frontend-slides

**用途**: 演示文稿制作

**使用场景**:
- 技术演讲
- 产品演示
- 架构图绘制

---

### design-system

**用途**: 设计系统构建

**核心组件**:
- 色彩系统
- 字体规范
- 组件库
- 文档站点

---

### canvas-design

**用途**: Canvas 画布设计

**使用场景**:
- 数据可视化
- 图表绘制
- 交互式图形

---

### ui-demo

**用途**: UI 演示制作

**用途**:
- 组件演示
- 交互原型
- 视觉验证

---

## Web 技术

### web-artifacts-builder

**用途**: Web 产物构建器

**功能**:
- HTML/CSS/JS 生成
- 原型构建
- 静态站点生成

---

## 相关技能

- `frontend-patterns` - 前端模式
- `frontend-design-direction` - 前端设计方向
- `frontend-slides` - 前端幻灯片
- `design-system` - 设计系统
- `canvas-design` - Canvas 设计
- `ui-demo` - UI 演示
