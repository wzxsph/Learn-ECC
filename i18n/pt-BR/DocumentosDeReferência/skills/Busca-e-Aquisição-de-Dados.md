# Busca-e-Aquisição-de-Dados技能

## 概述

Busca-e-Aquisição-de-Dados技能用于网络搜索、数据抓取、API 查询和信息检索。

## 核心技能

### exa-search

**用途**: Exa 语义搜索

**功能**:
- 语义网页搜索
- 内容提取
- 过滤和排序
- 实时更新

---

### search-first

**用途**: 搜索优先方法论

**核心原则**:
- 先搜索后实现
- 利用现有解决方案
- 避免重复造轮子

---

### data-scraper-agent

**用途**: 数据抓取代理

**使用场景**:
- 网页数据提取
- 结构化数据转换
- 批量采集
- 反爬虫策略

---

### iterative-retrieval

**用途**: 迭代检索模式

**方法**:
- 分层检索
- 相关性反馈
- 结果优化
- 上下文丰富

---

## MCP 服务器

### mcp-server-patterns

**用途**: MCP 服务器开发模式

**核心概念**:
- 工具定义
- 资源管理
- 提示模板
- 错误处理

---

### context7

**用途**: Context7 文档检索

**功能**:
- 库文档查询
- 代码示例获取
- API 引用
- 版本特定文档

---

## 相关技能

- `exa-search` - Exa 搜索
- `search-first` - 搜索优先
- `data-scraper-agent` - 数据抓取
- `iterative-retrieval` - 迭代检索
- `mcp-server-patterns` - MCP 服务器模式
- `context7` - Context7 文档
