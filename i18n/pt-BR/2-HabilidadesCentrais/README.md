# Stage 2: 核心能力 (Core Skills Mastery)

> 精通所有核心命令、掌握 Agent 协作、能定制 Hooks/Skills/Rules

## 课程概述

本阶段将带你深入掌握 Claude Code 的核心能力，包括 75+ 命令系统、6 类 Agent 协作、Hooks 定制、Skills 开发和 Rules 编写。完成本阶段后，你将能够完全掌控 Claude Code，实现高度定制化的开发工作流。

## 学习时长

**预计时间**: 1 周 (每天 2-3 小时)

## 前置要求

- 完成 [Stage 1: 入门基础](../1-Introdução/README.md)
- 已配置好 Claude Code 环境
- 熟悉基本的终端操作

## 学习目标

完成本阶段后，你将能够：

1. **命令系统精通**: 掌握所有 75+ 命令，能够根据场景选择最佳命令组合
2. **Agent 协作**: 理解 6 类 Agent 的职责，能组合使用完成复杂任务
3. **Hooks 定制**: 能够编写 PreToolUse/PostToolUse/Stop Hooks
4. **Skills 开发**: 学会创建和管理自定义 Skills，沉淀工作流模式
5. **Rules 编写**: 编写项目级规则，定制代码审查和质量标准

## 课程模块

| 模块 | 名称 | 内容 |

| 01 | [命令系统精通](./01-SistemaDeComandos/README.md) | 75+ 命令分类、使用场景、组合技巧 |

| 03 | [Hooks 定制](./03-PersonalizaçãoDeHooks/README.md) | Hook 类型、自定义 Hook、质量门禁 |
| 04 | [Skills 开发](./04-DesenvolvimentoDeSkills/README.md) | Skill 格式、创建方法、模式沉淀 |

## 学习建议

1. **循序渐进**: 按模块顺序学习，每个模块完成后再进入下一个
2. **动手实践**: 每个练习都要实际执行，不要只看不动手
3. **阅读教材**: 每个模块都附带了详细的技术文档链接
4. **记录笔记**: 记录自己的使用心得和最佳实践
