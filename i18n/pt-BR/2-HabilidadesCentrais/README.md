# Fase 2: Habilidades Essenciais (Domínio de Habilidades Centrais)

> Domine todos os comandos essenciais, aprenda colaboração de Agent, personalize Hooks/Skills/Rules

## Visão Geral do Curso

Esta fase levará você a dominar profundamente as habilidades centrais do Claude Code: sistema de 75+ comandos, colaboração de 6 tipos de Agents, customização de Hooks, desenvolvimento de Skills e criação de Rules. Após completar esta fase, você será capaz de controlar completamente o Claude Code e criar fluxos de trabalho de desenvolvimento altamente personalizados.

## Tempo de Aprendizado

**Tempo estimado**: 1 semana (2-3 horas por dia)

## Pré-requisitos

- [Fase 1: Introdução](../1-Introdução/README.md) concluída
- Ambiente Claude Code configurado
- Familiaridade com operações básicas de terminal

## Objetivos de Aprendizado

Após completar esta fase, você terá as seguintes habilidades:

1. **Domínio do sistema de comandos**: Será capaz de usar todos os 75+ comandos e criar as melhores combinações de comandos para cada cenário
2. **Colaboração de Agents**: Entenderá as responsabilidades dos 6 tipos de Agents e poderá usá-los em conjunto para completar tarefas complexas
3. **Customização de Hooks**: Será capaz de escrever Hooks PreToolUse/PostToolUse/Stop
4. **Desenvolvimento de Skills**: Aprenderá a criar e gerenciar Skills personalizados, acumulando padrões de workflow
5. **Criação de Rules**: Será capaz de escrever regras em nível de projeto e personalizar padrões de code review e qualidade

## Sugestões de Aprendizado

1. **Progresso gradual**: Estude cada módulo em sequência, não passe para o próximo antes de completar o atual
2. **Pratique**: Execute cada exercício realmente, não apenas leia
3. **Leia o material**: Cada módulo contém links para documentação técnica detalhada
4. **Tome notas**: Registre suas dicas de uso e melhores práticas
