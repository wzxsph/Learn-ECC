# 命令系统练习

## 练习目标

掌握 Claude Code 命令系统的创建、配置和调试。

## 练习内容

### 练习 1：创建基础命令
创建 `/mycommand` 命令，实现以下功能：
- 输出当前时间和项目路径
- 支持 `--verbose` 选项

### 练习 2：实现带参数命令
创建 `/search` 命令，实现：
- 接受搜索关键词参数
- 支持 `--limit` 参数控制结果数量
- 显示格式化输出

### 练习 3：命令调试
调试现有命令，添加错误处理和日志输出。

## 模板

```markdown
# 我的命令

## 用途
描述命令的主要功能。

## 使用方式
`/mycommand [选项] <参数>`

## 选项
- `--option`: 选项说明

## 示例
`/mycommand --option value`
```

## 验证
完成练习后，运行命令验证功能是否正常。
