# Проект 1: CLI-инструмент

Создайте командную утилиту с помощью ECC.

## Цели проекта

**Расчетное время**: 2-3 часа

Создайте инструмент `ecc-gen` для генерации компонентов проекта ECC (Agent, Skill, Hook).

## Требования к функциональности

### 1. Основные функции
- `ecc-gen agent <name>` - генерировать файл Agent
- `ecc-gen skill <name>` - генерировать файл Skill
- `ecc-gen hook <name>` - генерировать конфигурацию Hook

### 2. Интерактивные функции
- Генерация через интерактивные вопросы и ответы
- Выбор шаблона
- Указание выходного пути

### 3. Расширенные функции
- Пакетная генерация
- Пользовательские шаблоны
- Поддержка конфигурационных файлов

## Технический подход

### Структура каталога

```
ecc-gen/
├── src/
│   ├── commands/
│   │   ├── agent.js
│   │   ├── skill.js
│   │   └── hook.js
│   ├── generators/
│   │   └── templates/
│   └── index.js
├── package.json
└── README.md
```

### Используемые библиотеки
- `commander` - разбор параметров CLI
- `inquirer` - интерактивные вопросы
- `ejs` - движок шаблонов

## Критерии приемки

- [ ] Все три подкоманды работают корректно
- [ ] Сгенерированные шаблоны имеют правильный формат
- [ ] Поддержка указания выходного пути через `--output`
- [ ] Включены модульные тесты
- [ ] Документация полная

## Ключевые моменты обучения

- Разработка CLI-инструментов
- Проектирование системы шаблонов
- Обработка интерактивного ввода
- Модульная архитектура
