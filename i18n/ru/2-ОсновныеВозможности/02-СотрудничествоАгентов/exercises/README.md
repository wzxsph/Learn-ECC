# Сотрудничество AgentУпражнение

## УпражнениеЦель

学会在 Claude Code 中协调多个 Agent Завершить复杂Задача。

## Упражнение内容

### Упражнение 1：Параллельное выполнение задач
启动 2 个Параллельный Agent，分别Завершить：
- Agent A：Анализ структуры проекта
- Agent B：Проверить стандарты кода

### Упражнение 2：Цепные задачи
Реализовать 3 个 Agent 的Цепной вызов：
- Agent 1：Собрать требования
- Agent 2：Сгенерировать код
- Agent 3：Проверить код

### Упражнение 3：Агрегация результатов
将多个 Agent 的输出合并为统一报告。

## Шаблон

```markdown
# Сотрудничество Agent方案

## ЗадачаОписание
简述要Завершить的协作Задача。

## Участвующие Agent
| Agent | Роль | Задача |
|-------|------|------|
| agent-a | 分析 | xxx |
| agent-b | 执行 | xxx |

## Процесс выполнения
1. Запустить Agent A
2. Запустить Agent B
3. Агрегировать результаты

## Ожидаемый результат
ОписаниеФинальный формат вывода。
```

## Проверка
Проверить правильность агрегации выводов Agent。
