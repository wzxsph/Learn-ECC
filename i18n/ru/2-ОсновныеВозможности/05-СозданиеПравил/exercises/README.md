# Написание RulesУпражнение

## УпражнениеЦель

掌握 Claude Code 规则的创建和组织。

## Упражнение内容

### Упражнение 1：创建基础规则
创建 `my-rule.md`，包含：
- 规则目的Описание
- Сценарий применения定义
- 具体规范条款

### Упражнение 2：Реализовать多级规则
创建分层规则：
- 全局通用规则
- 项目特定规则
- 语言相关规则

### Упражнение 3：规则调试
调试规则优先级和覆盖逻辑。

## Rule Шаблон

```markdown
# 我的规则

## 目的
Описание规则要解决什么问题。

## 适用范围
Описание规则适用的Сценарий或文件Тип。

## 规范
- 规范 1
- 规范 2

## Проверка方式
Описание如何Проверка规则是否被遵守。
```

## Проверка
1. 触发规则Сценарий
2. 检查规则是否生效
3. Проверка优先级覆盖
