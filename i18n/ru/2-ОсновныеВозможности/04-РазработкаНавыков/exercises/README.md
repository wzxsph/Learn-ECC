# Разработка SkillsУпражнение

## УпражнениеЦель

学会创建和调试 Claude Code Skill 工作流。

## Упражнение内容

### Упражнение 1：创建基础 Skill
创建 `my-skill.md`，包含：
- When to Use 部分
- How It Works 部分
- 至少 2 个使用Пример

### Упражнение 2：Реализовать Skill 工作流
创建完整 Skill：
- 定义触发条件
- 编写执行Шаги
- 添加Обработка ошибок

### Упражнение 3：Skill 调试
调试现有 Skill，添加日志和Проверка逻辑。

## Skill Шаблон

```markdown
# 我的 Skill

## When to Use
Описание何时使用此 Skill。

## How It Works
Описание Skill 的工作原理和Процесс выполнения。

## Examples

### Пример 1
Описание第一个Сценарий использования。

### Пример 2
Описание第二个Сценарий использования。

## Примечание事项
- Примечание点 1
- Примечание点 2
```

## Проверка
1. 调用 Skill Проверка功能
2. 检查输出是否符合预期
