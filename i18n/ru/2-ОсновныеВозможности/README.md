# Этап 2: Основные возможности

> Мастерство всех основных команд, освоение сотрудничества агентов, умение настраивать Hooks/Skills/Rules

## Обзор курса

Этот этап проведет вас через глубокое освоение основных возможностей Claude Code, включая систему из 75+ команд, 6 типов агентов, настройку Hooks, разработку Skills и написание Rules. После завершения этого этапа вы сможете полностью управлять Claude Code и реализовывать высоко настроенные рабочие процессы разработки.

## Расчетное время

**Расчетное время**: 1 неделя (по 2-3 часа в день)

## Предварительные требования

- Завершить [Этап 1: Введение](../1-Введение/README.md)
- Claude Code окружение настроено
- Знакомы с основными операциями терминала

## Цели обучения

После завершения этапа вы сможете:

1. **Мастерство системы команд**: Освоить все 75+ команд, уметь выбирать оптимальные комбинации команд для конкретных сценариев
2. **Сотрудничество агентов**: Понять ответственность 6 типов агентов, уметь комбинировать их для выполнения сложных задач
3. **Настройка Hooks**: Уметь писать PreToolUse/PostToolUse/Stop Hooks
4. **Разработка Skills**: Научиться создавать и управлять своими Skills, накапливать паттерны рабочих процессов
5. **Написание Rules**: Писать правила уровня проекта, настраивать стандарты проверки кода и качества

1. **Последовательно**: Изучайте модули по порядку, переходите к следующему только после завершения текущего
2. **Практика**: Каждое упражнение выполняйте фактически, не просто читайте
3. **Чтение материалов**: Каждый модуль имеет ссылки на подробную техническую документацию
4. **Запись заметок**: Записывайте свои впечатления использования и лучшие практики
