# Проект 3: Автоматизация рабочего процесса

Создайте сквозную автоматизированную систему рабочих процессов.

## Цели проекта

**Расчетное время**: 2-3 часа

Спроектируйте и реализуйте автоматизированный движок рабочих процессов для обработки типичных задач разработки.

## Требования к функциональности

### 1. Определение рабочего процесса
- Определение в формате YAML
- Последовательное и параллельное выполнение шагов
- Условные ветвления
- Обработка циклов

### 2. Встроенные рабочие процессы
- **Проверка кода** - lint + type-check + test
- **Сборка и релиз** - build + test + deploy
- **Исправление проблем** - analyze + fix + verify

### 3. Мониторинг и оповещения
- Отслеживание статуса выполнения
- Уведомления об ошибках
- Статистика затраченного времени

## Технический подход

### Пример определения рабочего процесса

```yaml
name: code-check
steps:
  - name: lint
    command: npm run lint
  - name: type-check
    command: npm run type-check
  - name: test
    command: npm test
    requires: [lint, type-check]
```

### Интеграция Hooks
- PreToolUse - валидация параметров
- PostToolUse - валидация вывода
