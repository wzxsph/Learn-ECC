# Формат конфигурации

## hooks.json Обзор

ECC Система HooksJSON, ：

```json
{
  "$schema": "https://json.schemastore.org/claude-code-settings.json",
  "hooks": {
    "PreToolUse": [...],
    "PostToolUse": [...],
    "Stop": [...],
    "SessionStart": [...],
    "SessionEnd": [...],
    "PreCompact": [...],
    "PostToolUseFailure": [...]
  }
}
```

## 

|  | Тип | Описание |
|------|------|------|
| `$schema` | string | JSON Schema URL, Используется дляПроверка |
| `hooks` | object | Тип |

---

## 

Тип (PreToolUse, PostToolUse), ：

```json
{
  "matcher": "Bash|Edit|Write",
  "hooks": [
    {
      "type": "command",
      "command": "node scripts/hooks/my-hook.js",
      "async": false,
      "timeout": 30
    }
  ],
  "description": "Описание",
  "id": "unique-hook-id"
}
```

### matcher 

#### ТипОписание
matcherИспользуется дляТип, Поддерживать (`|`). 

#### 
| matcher |  |
|-----------|-----------------|
| `*` |  |
| `Bash` | Bash |
| `Edit` | Edit |
| `Write` | Write |
| `Read` | Read |
| `MultiEdit` | MultiEdit |
| `Bash\|Edit\|Write` | Bash, EditWrite |
| `Edit\|Write\|MultiEdit` | Edit, WriteMultiEdit |

#### Пример
```json
// 
"matcher": "*"

// Bash
"matcher": "Bash"

// 
"matcher": "Edit|Write|MultiEdit"
```

## Формат конфигурации

SessionStart, SessionEnd, PreCompact, ：

```json
{
  "description": "память",
  "events": [
    {
      "event": "SessionStart",
      "id": "session:start",
      "script": "scripts/hooks/session-start-bootstrap.js",
      "purpose": "Начатьобнаружение",
      "blocking": false
    },
    {
      "event": "PreCompact",
      "id": "pre:compact",
      "script": "scripts/hooks/pre-compact.js",
      "purpose": "",
      "blocking": false
    }
  ]
}
```

### 

|  | Описание |
|------|------|
| `event` | Тип (SessionStart, PreCompact, SessionEnd) |
| `id` |  |
| `script` |  |
| `purpose` | НазначениеОписание |
| `blocking` |  (false) |
