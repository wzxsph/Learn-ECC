## 

Требуется `~/.claude.json`  `mcpServers` ,  `YOUR_*_HERE` . 

**Пример -  GitHub ：**

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_your_actual_token"
      },
      "description": "GitHub operations - PRs, issues, repos"
    }
  }
}
```
