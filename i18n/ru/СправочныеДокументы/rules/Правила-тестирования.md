### AAA (Arrange-Act-Assert)

ТестированиеОбязательноAAA：

```typescript
test('', () => {
  // Arrange - Тестирование
  const vector1 = [1, 0, 0];
  const vector2 = [0, 1, 0];

  // Act - Тестирование
  const similarity = calculateCosineSimilarity(vector1, vector2);

  // Assert - Проверка
  expect(similarity).toBe(0);
});
```
