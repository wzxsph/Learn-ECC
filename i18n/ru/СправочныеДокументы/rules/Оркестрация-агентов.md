# Оркестрация агентов

##  Agent

 `~/.claude/agents/`:

| Agent | Назначение | Когда использовать |
|-------|------|----------|
| planner | Планирование реализации | функция,  |
| architect | Проектирование системы |  |
| tdd-guide | Разработка через тестирование | функция, bug  |
| code-reviewer | Код-ревью |  |
| security-reviewer | Анализ безопасности |  |
| build-error-resolver | Исправление ошибок сборки | неудача |
| e2e-runner | E2E тестирование |  |
| refactor-cleaner | Очистка мёртвого кода |  |
| doc-updater | Обновление документации |  |
| rust-reviewer | Rust Код-ревью | Rust  |
| harmonyos-app-resolver | Разработка HarmonyOS приложений | HarmonyOS/ArkTS  |

##  Agent

Подсказка：
1. функция →  **planner** agent
2. / →  **code-reviewer** agent
3. Bug функция →  **tdd-guide** agent
4.  →  **architect** agent

## Параллельное выполнение задач

,  Task ：

```markdown
# ：Параллельное выполнение
 3  agent ：
1. Agent 1: Анализ безопасности
2. Агент 2: Проверка производительности кэш-системы
3. Agent 3: Типпроверка

# Плохо: Ненужное последовательное выполнение
Сначала agent 1, потом agent 2, потом agent 3
```
