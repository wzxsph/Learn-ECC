# 后端与 API 技能

## Обзор

后端与 API 相关技能Используется для构建服务器端应用、API Спроектировать和数据库操作。

## 核心技能

### backend-patterns

**Назначение**: 后端开发Лучшие практики

**Основные концепции**:
- 分层架构（Controller → Service → Repository）
- 依赖注入
- Обработка ошибок策略
- 缓存模式
- 消息队列集成

---

### api-design

**Назначение**: API Спроектировать与RESTful 规范

**Основные концепции**:
- RESTful 路由Спроектировать
- 请求/响应格式
- 认证与授权
- 版本控制策略
- 限流与缓存

---

### api-connector-builder

**Назначение**: API 连接器构建工具

**Сценарий использования**:
- 第三方 API 集成
- Webhook 处理
- OAuth 流程Реализовать
- API 错误重试机制

---
