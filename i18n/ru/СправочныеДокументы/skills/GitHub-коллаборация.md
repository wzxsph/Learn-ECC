# GitHub 与协作技能

## Обзор

GitHub 与协作技能Используется для GitHub 操作、Код-ревью、PR Управлять和团队协作。

## 核心技能

### github-ops

**Назначение**: GitHub 操作与自动化

**功能**:
- Issue Управлять
- PR 创建与审查
- Код-ревью自动化
- GitHub Actions 工作流

---

### git-workflow

**Назначение**: Git 工作流规范

**Основные концепции**:
- 分支策略
- Commit 规范
- Merge 策略
- 冲突解决

---

### repo-scan

**Назначение**: 代码库扫描

**检查项**:
- 依赖漏洞
- 代码重复
- 圈复杂度
- 文档完整性

---

## Код-ревью

### click-path-audit

**Назначение**: 点击路径审计

**Сценарий использования**:
- 用户行为分析
- 转化率优化
- 漏斗分析

---

## 相关技能

- `github-ops` - GitHub 操作
- `git-workflow` - Git 工作流
- `repo-scan` - 代码库扫描
- `click-path-audit` - 点击路径审计
