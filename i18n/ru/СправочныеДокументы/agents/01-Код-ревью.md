## code-reviewer

### НазваниеНазначение
общий специалисткод-ревью, проактивнопроверять код, безопасностьподдерживаемость. после всех изменений кодаобязательно используется. 

### Описание
- качество кодапроверка (размер функций, глубина вложенности, дублирование кода)
- Уязвимости безопасностиобнаружение (SQL инъекция, XSS, Жёстко закодированные учётные данные)
- React/Next.js проверка
- Node.js/проверка
- Выявление проблем производительности
- Лучшие практики

### Сценарий применения
- 
- Pull Request 
- перед слиянием в общую ветку

### Список используемых инструментов
- Read: чтение содержимого файла
- Grep: поиск паттернов кода
- Glob: поиск файлов
- Bash:  git diff  lint 

### Сотрудничество с другими Agent
- Безопасностьпроблема Escalation  security-reviewer
- обнаружении ошибок сборки build-error-resolver
- Требуется refactor-cleaner
