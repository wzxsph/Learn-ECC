# Документация и исследования命令

## Обзор

Документация и исследования命令Используется для更新文档、Сгенерировать код地图和研究外部资源。

## 命令列表

### /update-docs

**Назначение**: 更新项目文档

**Описание**: 根据代码变更自动更新项目文档。

**更新内容**:
- README 文件
- API 文档
- 代码注释
- CHANGELOG

---

### /update-codemaps

**Назначение**: 重新生成 codemaps

**Описание**: 重新Сгенерировать код地图，Используется для代码库导航和理解。

**输出到**: `docs/CODEMAPS/`

---

### /ecc-guide

**Назначение**: ECC 用户指南

**Описание**: 显示 ECC 插件的完整用户指南。

---

## 相关命令

- `/update-docs` - 更新文档
- `/update-codemaps` - 更新代码地图
- `/ecc-guide` - 用户指南
