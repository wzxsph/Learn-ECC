## /go-test

**НазначениеОписание**: Go  TDD . Тестирование, ТестированиеРеализовать, Проверка 80%+ . 

**Способ использования**:
```
/go-test [функцияОписание]
```

**Сценарий использования**:
- Реализовать Go 
-  Go Тестирование
-  bug (неудачаТестирование)
-  Go TDD 

**Пример**:
```go
// Step 1: 
func ValidateEmail(email string) error {
    panic("not implemented")
}

// Step 2: Тестирование (RED)
tests := []struct {
    name    string
    email   string
    wantErr bool
}{
    {"simple email", "user@example.com", false},
    {"empty string", "", true},
    {"no at sign", "userexample.com", true},
}

// Step 3: Тестирование - Проверка FAIL

// Step 4: Реализовать (GREEN)
func ValidateEmail(email string) error {
    if email == "" {
        return ErrEmailEmpty
    }
    if !emailRegex.MatchString(email) {
        return ErrEmailInvalid
    }
    return nil
}

// Step 5: Проверка PASS

// Step 6: проверка
go test -cover ./...
// coverage: 100.0%
```

****:
```bash
go test -cover ./...                    # 
go test -coverprofile=coverage.out      # 
go tool cover -html=coverage.out        # 
go test -race -cover ./...              # обнаружение
```

-  Rust Тестирование
-  bug (неудачаТестирование)

**Тестирование**:
```rust
// Тестирование
#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn adds_two_numbers() {
        assert_eq!(add(2, 3), 5);
    }

    #[test]
    fn handles_error() -> Result<(), Box<dyn std::error::Error>> {
        let result = parse_config(r#"port = 8080"#)?;
        assert_eq!(result.port, 8080);
        Ok(())
    }
}

// Тестирование (rstest)
#[rstest]
#[case("hello", 5)]
#[case("", 0)]
#[case("rust", 4)]
fn test_string_length(#[case] input: &str, #[case] expected: usize) {
    assert_eq!(input.len(), expected);
}

// Тестирование
#[tokio::test]
async fn fetches_data_successfully() {
    let client = TestClient::new().await;
    let result = client.get("/data").await;
    assert!(result.is_ok());
}

// Тестирование (proptest)
proptest! {
    #[test]
    fn encode_decode_roundtrip(input in ".*") {
        let encoded = encode(&input);
        let decoded = decode(&encoded).unwrap();
        assert_eq!(input, decoded);
    }
}
```
