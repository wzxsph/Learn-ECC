# Управление проектом命令

## Обзор

Управление проектом命令Используется для项目配置、基础设施设置和工具链Управлять。

## 命令列表

### /projects

**Назначение**: 列出已知项目 + instinct 统计

**Описание**: 列出用户所有已知项目及其相关的 instinct 统计信息。

---

### /harness-audit

**Назначение**: 审计 agent harness 配置

**Описание**: 检查和审计 Claude Code 的 harness 配置。

**检查内容**:
- 工具链配置
- 代理设置
- MCP сервер状态
- 规则配置

---

### /model-route

**Назначение**: 路由Задача到合适模型

**Описание**: 根据ЗадачаТип选择最合适的模型（Haiku/Sonnet/Opus）。

**模型选择指南**:
- **Haiku**: 轻量级Задача、代码生成、频繁调用
- **Sonnet**: 主要开发工作、复杂编码Задача
- **Opus**: 架构决策、深度推理、研究分析

---

### /pm2

**Назначение**: PM2 进程Управлять器初始化

**Описание**: 初始化 PM2 进程Управлять器配置。

---

### /setup-pm

**Назначение**: 配置包Управлять器

**Описание**: 配置项目使用的包Управлять器。

**Поддерживать的包Управлять器**:
- npm
- pnpm
- yarn
- bun

**可Через** `CLAUDE_PACKAGE_MANAGER` **环境变量配置**

---

### /project-init

**Назначение**: 项目初始化

**Описание**: 初始化新项目的标准结构和配置。

---

## 相关命令

- `/projects` - 列出项目
- `/harness-audit` - 审计配置
- `/model-route` - 模型路由
