# Agent 速查

## 内置 Agent

### Планирование
| Agent | Назначение | Особенности |
|-------|------|------|
| `planner` | [Планирование реализации](../СправочныеДокументы/agents/6.%20Архитектура.md) | 分解Задача、生成Шаги |
| `architect` | [Проектирование системы](../СправочныеДокументы/agents/6.%20Архитектура.md) | 架构决策、Выбор технологий |

### 审查类
| Agent | Назначение | Особенности |
|-------|------|------|
| `code-reviewer` | Код-ревью | 质量把关、模式检查 |
| `security-reviewer` | Безопасность审查 | 漏洞检测、OWASP |
| `rust-reviewer` | Rust 审查 | 生命周期、借用检查 |

### 开发类
| Agent | Назначение | Особенности |
|-------|------|------|
| `tdd-guide` | [TDD指导](../СправочныеДокументы/agents/1.%20Код-ревью.md) | Тестирование先行、增量重构 |
| `build-error-resolver` | [Исправление сборки](../СправочныеДокументы/agents/2.%20Исправление сборки.md) | 错误定位、修复建议 |

### 运维类
| Agent | Назначение | Особенности |
|-------|------|------|
| `e2e-runner` | [E2EТестирование](../СправочныеДокументы/agents/1.%20Код-ревью.md) | 关键路径、自动化 |
| `refactor-cleaner` | [重构清理](../СправочныеДокументы/agents/1.%20Код-ревью.md) | 死代码移除 |

## Agent 调用方式

```
/[agent-name]                    # Прямой вызов
/[agent-name] [parameters]       # 带参数
/[agent-name] --option value    # 选项参数
```

## 自定义 Agent

Agent 文件格式：`agents/*.md` + YAML frontmatter

```yaml
---
name: my-agent
description: 我的自定义 Agent
tools: [Read, Bash, Edit]
model: sonnet
---
```
