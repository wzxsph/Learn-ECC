# Агенты速查

## Встроенные Агенты

### Планирование
| Agent | Назначение | Особенности |
|-------|-----------|-------------|
| `planner` | [Планирование реализации](../СправочныеДокументы/agents/6.%20Архитектура.md) | Декомпозиция задач, генерация шагов |
| `architect` | [Проектирование системы](../СправочныеДокументы/agents/6.%20Архитектура.md) | Архитектурные решения, выбор технологий |

### Проверка
| Agent | Назначение | Особенности |
|-------|-----------|-------------|
| `code-reviewer` | Проверка кода | Контроль качества, проверка паттернов |
| `security-reviewer` | Проверка безопасности | Обнаружение уязвимостей, OWASP |
| `rust-reviewer` | Проверка Rust | Жизненный цикл, проверка заимствований |

### Разработка
| Agent | Назначение | Особенности |
|-------|-----------|-------------|
| `tdd-guide` | [TDD руководство](../СправочныеДокументы/agents/1.%20Код-ревью.md) | Тестирование в первую очередь, инкрементное рефакторинг |
| `build-error-resolver` | [Исправление сборки](../СправочныеДокументы/agents/2.%20Исправление сборки.md) | Локализация ошибок, рекомендации по исправлению |

### Операции
| Agent | Назначение | Особенности |
|-------|-----------|-------------|
| `e2e-runner` | [E2E тестирование](../СправочныеДокументы/agents/1.%20Код-ревью.md) | Критические пути, автоматизация |
| `refactor-cleaner` | [Рефакторинг-очистка](../СправочныеДокументы/agents/1.%20Код-ревью.md) | Удаление мертвого кода |

## Способы вызова Agent

```
/[agent-name]                    # Прямой вызов
/[agent-name] [parameters]       # С параметрами
/[agent-name] --option value    # Параметры опций
```

## Пользовательские Агенты

Формат файла Agent: `agents/*.md` + YAML frontmatter

```yaml
---
name: my-agent
description: Мой пользовательский Agent
tools: [Read, Bash, Edit]
model: sonnet
---
```
