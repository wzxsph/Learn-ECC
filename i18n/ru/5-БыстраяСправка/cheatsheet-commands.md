# 命令速查

## 斜杠命令

| 命令 | 功能 | Сценарий использования |
|------|------|----------|
| `/tdd` | Разработка через тестирование工作流 | 新功能开发 |
| `/plan` | Реализовать计划生成 | 复杂Задача规划 |
| `/code-review` | [Код-ревью](../СправочныеДокументы/commands/01-Основной рабочий процесс.md) | Качество кода检查 |
| `/build-fix` | [Исправление сборки](../СправочныеДокументы/commands/04-Исправление сборки.md) | 构建失败时 |
| `/learn` | 从会话提取模式 | 知识沉淀 |
| `/skill-create` | 从 Git 生成 Skill | 经验复用 |
| `/test` | [Связанное с тестированием命令](../СправочныеДокументы/commands/02-Связанное с тестированием.md) | Связанное с тестированием |

## 脚本命令

| 脚本 | 功能 |
|------|------|
| `node scripts/hooks/run-with-flags.js` | 带标志位运行钩子 |
| `node tests/run-all.js` | 运行全量Тестирование |
| `node scripts/lib/utils.js` | Библиотека вспомогательных функций |

## Agent 命令

| Agent | 功能 |
|-------|------|
| `/planner` | Задача规划 |
| `/code-reviewer` | Код-ревью |
| `/tdd-guide` | TDD 指导 |
| `/security-reviewer` | Безопасность审查 |
| `/build-error-resolver` | Исправление сборки |

## 全局参数

| 参数 | Описание |
|------|------|
| `--model` | 指定模型 |
| `--no-input` | 非交互模式 |
| `--output` | 输出格式 |

## Hook 环境变量

| 变量 | Описание |
|------|------|
| `ECC_HOOK_PROFILE` | 钩子配置 |
| `ECC_DISABLED_HOOKS` | 禁用钩子列表 |
