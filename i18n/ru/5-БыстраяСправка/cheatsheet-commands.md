# Команды速查

## Слэш-команды

| Команда | Функция | Сценарий использования |
|---------|---------|------------------------|
| `/tdd` | Разработка через тестирование | Разработка новой функциональности |
| `/plan` | Создание плана реализации | Планирование сложных задач |
| `/code-review` | [Проверка кода](../СправочныеДокументы/commands/01-核心工作流.md) | Проверка качества кода |
| `/build-fix` | [Исправление сборки](../СправочныеДокументы/commands/04-构建修复.md) | При сбое сборки |
| `/learn` | Извлечение паттернов из сессии | Накопление знаний |
| `/skill-create` | Создание Skill из Git | Повторное использование опыта |
| `/test` | [Команды, связанные с тестированием](../СправочныеДокументы/commands/02-测试相关.md) | Тестирование |

## Скриптовые команды

| Скрипт | Функция |
|--------|---------|
| `node scripts/hooks/run-with-flags.js` | Запуск хуков с флагами |
| `node tests/run-all.js` | Запуск всех тестов |
| `node scripts/lib/utils.js` | Библиотека вспомогательных функций |

## Команды Agent

| Agent | Функция |
|-------|---------|
| `/planner` | Планирование задач |
| `/code-reviewer` | Проверка кода |
| `/tdd-guide` | TDD руководство |
| `/security-reviewer` | Проверка безопасности |
| `/build-error-resolver` | Исправление сборки |

## Глобальные параметры

| Параметр | Описание |
|----------|----------|
| `--model` | Указать модель |
| `--no-input` | Неинтерактивный режим |
| `--output` | Формат вывода |

## Переменные окружения Hook

| Переменная | Описание |
|------------|----------|
| `ECC_HOOK_PROFILE` | Конфигурация хуков |
| `ECC_DISABLED_HOOKS` | Список отключенных хуков |
