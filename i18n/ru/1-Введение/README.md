# Pervyy etap: Vvedeniye

Etap dizayn edildi dlya pol'zovateley s nul'nym urovnem, pomozhet vam bystro ponyat' ECC i nachat' ispol'zovaniye.

## Tseli obucheniya

Posle zaversheniya etapa vy smozhete:
- Ponyat' osnovnyye kontseptsii i stsenarii primeneniya ECC
- Vypolnit' ustanovku i bazovuyu konfiguratsiyu Claude Code
- Uvpered' ispol'zovat' chasto ispol'zuyemyye komandy
- Samostoyatel'no vypolnyat' prostyye zadachi razrabotki

## Raschetnoye vremya

2-3 chasa

## Predvaritel'nyye trebovaniya

- Osnovy komandnoy stroki
- Bazovyye znaniya JavaScript/Node.js (dostatochno ponyatiye)
- Osnovnyye kontseptsii Git
