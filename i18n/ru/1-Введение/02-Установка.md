# 02 - Установка

## Требования к среде

- Node.js >= 18
- Один из npm / pnpm / yarn / bun
- Claude Code CLI v2.1.0+ установлен

## Способы установки

### Способ 1: Установка плагина (Рекомендуется)

#### 1. Добавьте маркетплейс плагинов

```bash
/plugin marketplace add https://github.com/affaan-m/ECC
```

#### 2. Установите плагин

```bash
/plugin install ecc@ecc
```

#### 3. Установите Rules (плагин не устанавливает автоматически)

```bash
# Клонируйте репозиторий
git clone https://github.com/affaan-m/ECC.git
cd ECC

# Скопируйте правила в пользовательский каталог
mkdir -p ~/.claude/rules/ecc
cp -r rules/common ~/.claude/rules/ecc/
# Выберите ваш стек технологий
cp -r rules/typescript ~/.claude/rules/ecc/   # или python, golang, swift, php
```

### Способ 2: Ручная установка

Если вам нужен более точный контроль:

```bash
# Клонируйте репозиторий
git clone https://github.com/affaan-m/ECC.git
cd ECC

# Установите зависимости
npm install        # или pnpm install | yarn install | bun install

# Установите agents
cp agents/*.md ~/.claude/agents/

# Установите skills (основная поверхность рабочих процессов)
mkdir -p ~/.claude/skills/ecc
cp -r .agents/skills/* ~/.claude/skills/ecc/

# Установите rules
mkdir -p ~/.claude/rules/ecc
cp -r rules/common ~/.claude/rules/ecc/
cp -r rules/typescript ~/.claude/rules/ecc/

# Установите commands (опционально, ECC переходит на skills)
mkdir -p ~/.claude/commands
cp commands/*.md ~/.claude/commands/
```

## Проверка установки

```bash
# Проверьте, успешно ли установлен плагин
/plugin list ecc@ecc

# Проверьте доступные команды
/ecc:plan
/ecc:code-review
```

## Базовая настройка

### Настройка менеджера пакетов

ECC автоматически определяет предпочитаемый вами менеджер пакетов, но вы также можете настроить вручную:

```bash
# Через переменную окружения
export CLAUDE_PACKAGE_MANAGER=pnpm

# или через команду
/setup-pm
```

Приоритет: Переменная окружения > Конфигурация проекта > package.json > lock файл > Глобальная конфигурация

### Управление временем выполнения Hook

```bash
# Строгость Hook (по умолчанию: standard)
export ECC_HOOK_PROFILE=standard

# Отключить определенные Hooks
export ECC_DISABLED_HOOKS="pre:bash:tmux-reminder,post:edit:typecheck"

# Лимит контекста SessionStart (по умолчанию: 8000 символов)
export ECC_SESSION_START_MAX_CHARS=4000
```

### Настройки модели

Рекомендуется настроить в `~/.claude/settings.json`:

```json
{
  "model": "sonnet",
  "env": {
    "MAX_THINKING_TOKENS": "10000",
    "CLAUDE_AUTOCOMPACT_PCT_OVERRIDE": "50"
  }
}
```
