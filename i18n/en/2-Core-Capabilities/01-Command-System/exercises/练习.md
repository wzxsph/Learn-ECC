# Command System Exercises

## Exercise Objectives

Master creating, configuring, and debugging the Claude Code command system.

## Exercise Content

### Exercise 1: Create Basic Command
Create `/mycommand` command implementing:
- Output current time and project path
- Support `--verbose` option

### Exercise 2: Implement Command with Parameters
Create `/search` command implementing:
- Accept search keyword parameter
- Support `--limit` parameter to control result count
- Display formatted output

### Exercise 3: Command Debugging
Debug existing commands, add error handling and log output.

## Template

```markdown
# My Command

## Purpose
Describe the main function of the command.

## Usage
`/mycommand [options] <argument>`

## Options
- `--option`: Option description

## Example
`/mycommand --option value`
```

## Verification

After completing exercises, run commands to verify functionality.
