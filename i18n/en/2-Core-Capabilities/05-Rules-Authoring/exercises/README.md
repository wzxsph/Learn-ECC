# Rules Authoring Exercises

## Exercise Objectives

Master creating and organizing Claude Code rules.

## Exercise Content

### Exercise 1: Create Basic Rule
Create `my-rule.md` containing:
- Rule purpose description
- Applicable scope definition
- Specific rule clauses

### Exercise 2: Implement Multi-level Rules
Create layered rules:
- Global common rules
- Project-specific rules
- Language-related rules

### Exercise 3: Rule Debugging
Debug rule priority and override logic.

## Rule Template

```markdown
# My Rule

## Purpose
Describe what problem the rule solves.

## Scope
Describe scenarios or file types the rule applies to.

## Rules
- Rule 1
- Rule 2

## Verification
Describe how to verify the rule is followed.
```

## Verification
1. Trigger rule scenarios
2. Check if rule takes effect
3. Verify priority override
