# Stage 2: Core Capabilities (Core Skills Mastery)

> Master all core commands, understand Agent collaboration, customize Hooks/Skills/Rules

## Course Overview

This stage will take you deep into mastering Claude Code's core capabilities, including 75+ command system, 6 types of Agent collaboration, Hooks customization, Skills development, and Rules authoring. After completing this stage, you will have complete control over Claude Code and achieve highly customized development workflows.

## Learning Duration

**Estimated Time**: 1 week (2-3 hours daily)

## Prerequisites

- Completed [Stage 1: Getting Started](../1-Getting-Started/README.md)
- Claude Code environment configured
- Familiar with basic terminal operations

## Learning Objectives

After completing this stage, you will be able to:

1. **Command System Mastery**: Master all 75+ commands, select optimal command combinations for scenarios
2. **Agent Collaboration**: Understand responsibilities of 6 types of Agents, combine them for complex tasks
3. **Hooks Customization**: Write PreToolUse/PostToolUse/Stop Hooks
4. **Skills Development**: Create and manage custom Skills,沉淀 workflow patterns
5. **Rules Authoring**: Write project-level rules, customize code review and quality standards
