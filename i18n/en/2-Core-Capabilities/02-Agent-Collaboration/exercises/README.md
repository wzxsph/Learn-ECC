# Agent Collaboration Exercises

## Exercise Objectives

Learn to coordinate multiple Agents in Claude Code to complete complex tasks.

## Exercise Content

### Exercise 1: Parallel Task Execution
Launch 2 parallel Agents, each completing:
- Agent A: Analyze project structure
- Agent B: Check code standards

### Exercise 2: Chain Tasks
Implement 3-Agent chain invocation:
- Agent 1: Collect requirements
- Agent 2: Generate code
- Agent 3: Review code

### Exercise 3: Result Aggregation
Merge outputs from multiple Agents into a unified report.

## Template

```markdown
# Agent Collaboration Plan

## Task Description
Briefly describe the collaboration task to complete.

## Participating Agents
| Agent | Role | Task |
|-------|------|------|
| agent-a | Analysis | xxx |
| agent-b | Execution | xxx |

## Execution Flow
1. Launch Agent A
2. Launch Agent B
3. Aggregate results

## Expected Output
Describe the final output format.
```

## Verification

Check that Agent outputs are correctly aggregated.
