# Skills Development Exercises

## Exercise Objectives

Learn to create and debug Claude Code Skill workflows.

## Exercise Content

### Exercise 1: Create Basic Skill
Create `my-skill.md` containing:
- When to Use section
- How It Works section
- At least 2 usage examples

### Exercise 2: Implement Skill Workflow
Create complete Skill:
- Define trigger conditions
- Write execution steps
- Add error handling

### Exercise 3: Skill Debugging
Debug existing Skill, add logging and verification logic.

## Skill Template

```markdown
# My Skill

## When to Use
Describe when to use this Skill.

## How It Works
Describe how the Skill works and its execution flow.

## Examples

### Example 1
Describe the first usage scenario.

### Example 2
Describe the second usage scenario.

## Notes
- Note 1
- Note 2
```

## Verification
1. Invoke Skill to verify functionality
2. Check if output matches expectations
