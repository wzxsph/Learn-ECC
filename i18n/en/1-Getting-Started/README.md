# Stage 1: Getting Started

This stage is designed for users with no prior experience, helping you quickly understand ECC and get started.

## Learning Objectives

After completing this stage, you will be able to:
- Understand ECC's basic concepts and use cases
- Complete Claude Code installation and basic configuration
- Master common commands
- Independently complete simple development tasks

## Estimated Duration

2-3 hours

## Prerequisites

- Command line basics
- JavaScript/Node.js basics (understanding is sufficient)
- Git basic concepts
