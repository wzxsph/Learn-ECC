# 02 - Autonomous Agents

Learn how to build agent systems that can autonomously make decisions and execute tasks.

## What are Autonomous Agents?

Autonomous agents are intelligent systems that can:
- Autonomously plan task steps
- Adjust strategies based on environmental feedback
- Continuously execute until goals are achieved

> Reference: [Automation & Scripting](../Reference-Docs/skills/自动化与脚本.md) | [Code Review Agents](../Reference-Docs/agents/1.%20代码审查类.md)

## Learning Objectives

- Understand autonomous agent architecture
- Design agent decision flows
- Implement agent execution loops
- Handle exceptions and recovery

## Core Components

### 1. Planner
Decomposes complex tasks into executable steps.

### 2. Executor
Invokes tools to execute specific operations.

### 3. Evaluator
Determines if tasks are complete and evaluates execution results.

### 4. Memory
Stores conversation history and context.

## Agent Patterns

### Reactive Agents
Respond directly to environmental changes based on current state.

### Goal-Oriented Agents
Plan and execute around clear goals.

### Learning Agents
Improve decision strategies through experience.
