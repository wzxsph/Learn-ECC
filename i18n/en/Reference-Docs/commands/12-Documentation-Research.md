# Documentation and Research Commands

## Overview

Documentation and research commands are used for updating documentation, generating codemaps, and researching external resources.

## Command List

### /update-docs

**Purpose**: Update project documentation

**Description**: Automatically update project documentation based on code changes.

**Updates**:
- README files
- API documentation
- Code comments
- CHANGELOG

---

### /update-codemaps

**Purpose**: Regenerate codemaps

**Description**: Regenerate code maps for codebase navigation and understanding.

**Output to**: `docs/CODEMAPS/`

---

### /ecc-guide

**Purpose**: ECC user guide

**Description**: Display complete user guide for the ECC plugin.

---

## Related Commands

- `/update-docs` - Update documentation
- `/update-codemaps` - Update codemaps
- `/ecc-guide` - User guide
