# Deployment and DevOps Skills

## Overview

Deployment and DevOps skills are used for containerization, infrastructure as code, continuous integration, and deployment automation.

## Core Skills

### docker-patterns

**Purpose**: Docker container best practices

**Core concepts**:
- Multi-stage builds
- Minimizing image size
- Health checks
- Log management
- Resource limits

---

### deployment-patterns

**Purpose**: Deployment strategies and patterns

**Core concepts**:
- Blue-green deployment
- Canary releases
- Rolling updates
- Rollback strategies
- Zero-downtime deployment

---

### production-audit

**Purpose**: Production environment audit

**Checklist**:
- Security configuration
- Performance metrics
- Availability checks
- Monitoring alerting verification

---

### production-scheduling

**Purpose**: Production scheduling management

**Use cases**:
- Scheduled task management
- Batch job processing
- Maintenance window planning

---

### canary-watch

**Purpose**: Canary release monitoring

**Features**:
- Traffic split monitoring
- Error rate tracking
- Performance comparison
- Automatic rollback triggers

---

## Infrastructure

### network-config-validation

### network-interface-health
