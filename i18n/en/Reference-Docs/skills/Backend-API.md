# Backend and API Skills

## Overview

Backend and API skills are used for building server-side applications, API design, and database operations.

## Core Skills

### backend-patterns

**Purpose**: Backend development best practices

**Core concepts**:
- Layered architecture (Controller → Service → Repository)
- Dependency injection
- Error handling strategies
- Caching patterns
- Message queue integration

---

### api-design

**Purpose**: API design and RESTful standards

**Core concepts**:
- RESTful route design
- Request/response format
- Authentication and authorization
- Versioning strategies
- Rate limiting and caching

---

### api-connector-builder

**Purpose**: API connector builder

**Use cases**:
- Third-party API integration
- Webhook handling
- OAuth flow implementation
- API error retry mechanisms

---
