# Monitoring and Observability Skills

## Overview

Monitoring and observability skills are used for system monitoring, log management, tracing, and alerting configuration.

## Core Skills

### observability-readiness

**Purpose**: Observability readiness check

**Checklist**:
- Log configuration
- Metrics exposure
- Tracing integration
- Alert rules

---

### observability-designer

**Purpose**: Observability architecture design

**Design content**:
- Metrics system
- Log architecture
- Tracing solution
- Correlation analysis

---
