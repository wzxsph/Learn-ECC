# Stage 4: Practical Projects

Consolidate learned knowledge through real projects.

## Learning Objectives

After completing this stage, you will be able to:
- Build complete CLI tools from scratch
- Implement automated code review pipelines
- Design end-to-end automated workflows
- Build multi-Agent collaboration platforms

## Estimated Duration

10-15 hours

## Prerequisites

- Completed [Stage 2: Core Capabilities](../2-Core-Capabilities/README.md)
- Proficient in at least one scripting language
- Project development experience

## Learning Content

| Project | Content | Difficulty |
|---------|---------|------------|
| [Project 1: CLI Tool](./project-1-cli.md) | Build command-line tool | Basic |
| [Project 2: Code Review](./project-2-review.md) | Automated review pipeline | Intermediate |
| [Project 3: Workflow Automation](./project-3-auto.md) | End-to-end automation | Intermediate |
| [Project 4: Agent System](./project-4-agent.md) | Multi-Agent collaboration platform | Advanced |

## Suggestions

Choose one project you're interested in to深入 complete, rather than dabbling in all projects.
Recommended order: CLI Tool → Code Review → Workflow Automation → Agent System (increasing difficulty)

## Next Steps

After completing practical projects, proceed to [Stage 5: Quick Reference](../5-Quick-Reference/README.md) for common commands and patterns.
