# 第3段階：熟練への道

本段階では ECC の上級機能と専門家レベルの使い方を探求します。

## 学習目標

本段階を完了すると、以下ができる：
- MCP（モデルコンテキストプロトコル）拡張を 개발
- 自律エージェントシステムを構築
- 上級設計モードとベストプラクティスを応用

## 予想所要時間

6-8時間

## 前提要件

- [第2段階：コア能力](../2-コア能力/README.md) を完了
- 豊富なプロジェクト開発経験
- システムアーキテクチャ設計を理解

## 学習内容

| 章 | 内容 |
|------|------|
| [01-MCP開発](./01-MCP開発.md) | モデルコンテキストプロトコル、拡張開発 |
| [02-自律エージェント](./02-自律エージェント.md) | 自律 Agent、決定システム |
| [03-高度なモード](./03-高度なモード.md) | 設計モード、ベストプラクティス |

## 次へ

熟練への道段階を完了すると、[第4段階：実践プロジェクト](../4-実践プロジェクト/README.md)に進んで所学を実践に適用します。
