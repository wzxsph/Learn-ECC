# Hooks システム

## Hook タイプ

- **PreToolUse**: ツール実行前（検証、参数修改）
- **PostToolUse**: ツール実行后（自動フォーマット化、検査）
- **Stop**: セッション終了时（最终検証）

## 自動受け入れ权限

谨慎使用：
- 为可信的、定义明确的計画あり効化
- 为探索性作業無効化
- 切勿使用 dangerously-skip-permissions 标志
- 在 `~/.claude.json` 中設定 `allowedTools`

## TodoWrite ベストプラクティス

使用 TodoWrite ツール：
- トレース多ステップタスク的进度
- 検証对指令的理解
- あり効化リアルタイム指导
- 表示粒度实施ステップ

Todo 列表揭示：
- 乱序ステップ
- 缺失プロジェクト
- 多余的不必要プロジェクト
- エラー的粒度
- 误解的需求
