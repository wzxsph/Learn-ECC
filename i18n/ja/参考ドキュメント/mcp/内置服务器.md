# 内置 MCP サーバー

ECC プロジェクト在 `mcp-configs/mcp-servers.json` 中预置了多个常用 MCP サーバー設定。

## 按機能分類

### 開発ツール

| サーバー | タイプ | 描述 |
|--------|------|------|
| `github` | process | GitHub 操作 - PRs, issues, repos |
| `jira` | process | Jira 問題トレース - 搜索、作成、更新、评论、流转問題 |
| `confluence` | process | Confluence Cloud 統合 - 搜索页面、取得内容、探索スペース |
| `filesystem` | process | ファイルシステム操作（需設定路径） |
| `memory` | process | 跨セッション持久化メモリ |
| `omega-memory` | process | 增强版持久化プロキシメモリ，サポート语义搜索、多プロキシ协作和知识图谱 |
| `sequential-thinking` | process | 链式思维推論 |

### デプロイ与インフラ

| サーバー | タイプ | 描述 |
|--------|------|------|
| `vercel` | http | Vercel デプロイ和プロジェクト |
| `railway` | process | Railway デプロイ |
| `cloudflare-docs` | http | Cloudflare ドキュメント搜索 |
| `cloudflare-workers-builds` | http | Cloudflare Workers ビルド |
| `cloudflare-workers-bindings` | http | Cloudflare Workers 绑定 |
| `cloudflare-observability` | http | Cloudflare 可観測性/ログ |

### データベース

| サーバー | タイプ | 描述 |
|--------|------|------|
| `supabase` | process | Supabase データベース操作 |
| `clickhouse` | http | ClickHouse 分析查询 |

### AI 与機械学習

| サーバー | タイプ | 描述 |
|--------|------|------|
| `fal-ai` | process | fal.ai モデル的 AI 图片/视频/音频生成 |
| `exa-web-search` | process | 通過 Exa API 进行网页搜索和研究 |
| `context7` | process | リアルタイムドキュメント查询 - 与 /docs コマンド和 documentation-lookup skill 配合使用 |

## 快速あり効化

コピー必要的サーバー設定到 `~/.claude.json` 的 `mcpServers` 节，置換 `YOUR_*_HERE` 占位符为实际值。

**サンプル - あり効化 GitHub サーバー：**

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_your_actual_token"
      },
      "description": "GitHub operations - PRs, issues, repos"
    }
  }
}
```

## 环境変数要件

| サーバー | 必需的环境変数 |
|--------|----------------|
| `github` | `GITHUB_PERSONAL_ACCESS_TOKEN` |
| `jira` | `JIRA_URL`, `JIRA_EMAIL`, `JIRA_API_TOKEN` |
| `firecrawl` | `FIRECRAWL_API_KEY` |
| `supabase` | `--project-ref` 参数 |
| `exa-web-search` | `EXA_API_KEY` |
| `fal-ai` | `FAL_KEY` |
| `browserbase` | `BROWSERBASE_API_KEY` |
| `browser-use` | `x-browser-use-api-key` リクエスト头 |
| `confluence` | `CONFLUENCE_BASE_URL`, `CONFLUENCE_EMAIL`, `CONFLUENCE_API_TOKEN` |
| `evalview` | `OPENAI_API_KEY`（オプション） |
