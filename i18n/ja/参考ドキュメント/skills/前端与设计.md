# フロントエンドとデザイン技能

## 概要

フロントエンドとデザイン技能用于ビルド现代 Web 应用、UI/UX 設計和フロントエンドアーキテクチャ。

## コア技能

### frontend-patterns

**用途**: フロントエンド開発ベストプラクティス

**コア概念**:
- コンポーネントアーキテクチャ
- 状態管理
- パフォーマンス最適化
- 応答式設計
- なし障碍访问

---

### frontend-design-direction

**用途**: フロントエンド設計方向指导

**涵盖内容**:
- 設計システム
- 設計令牌
- 主题化
- 品牌一致性

---

### frontend-slides

**用途**: 演示文稿制作

**使用场景**:
- 技術演讲
- 产品演示
- アーキテクチャ图绘制

---

### design-system

**用途**: 設計システムビルド

**コアコンポーネント**:
- 色彩システム
- 字体仕様
- コンポーネント库
- ドキュメント站点

---

### canvas-design

**用途**: Canvas 画布設計

**使用场景**:
- データ可视化
- 图表绘制
- 交互式图形

---

### ui-demo

**用途**: UI 演示制作

**用途**:
- コンポーネント演示
- 交互原型
- 视觉検証

---

## Web 技術

### web-artifacts-builder

**用途**: Web 产物ビルド器

**機能**:
- HTML/CSS/JS 生成
- 原型ビルド
- 静态站点生成

---

## 関連スキル

- `frontend-patterns` - フロントエンドパターン
- `frontend-design-direction` - フロントエンド設計方向
- `frontend-slides` - フロントエンド幻灯片
- `design-system` - 設計システム
- `canvas-design` - Canvas 設計
- `ui-demo` - UI 演示
