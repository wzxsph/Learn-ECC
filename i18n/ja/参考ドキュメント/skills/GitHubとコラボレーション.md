# GitHub 與协作技能

## 概要

GitHub 與协作技能使用 GitHub 操作、コードレビュー、PR 管理和團隊协作。

## コア技能

### github-ops

**用途**: GitHub 操作與自動化

**機能**:
- Issue 管理
- PR 作成與レビュー
- コードレビュー自動化
- GitHub Actions ワークフロー

---

### git-workflow

**用途**: Git ワークフロー仕様

**コア概念**:
- ブランチ戦略
- Commit 仕様
- Merge 戦略
- 冲突解決

---

### repo-scan

**用途**: コードベース扫描

**チェック項目**:
- 依賴漏洞
- コード重复
- 圈複雜度
- ドキュメント完整性

---

## コードレビュー

### click-path-audit

**用途**: 点击パス審計

**使用场景**:
- ユーザー行為分析
- 转化率最適化
- 漏斗分析

---

## 関連スキル

- `github-ops` - GitHub 操作
