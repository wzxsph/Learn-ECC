# GitHub 与协作技能

## 概要

GitHub 与协作技能用于 GitHub 操作、コードレビュー、PR 管理和团队协作。

## コア技能

### github-ops

**用途**: GitHub 操作与自動化

**機能**:
- Issue 管理
- PR 作成与レビュー
- コードレビュー自動化
- GitHub Actions ワークフロー

---

### git-workflow

**用途**: Git ワークフロー仕様

**コア概念**:
- ブランチ戦略
- Commit 仕様
- Merge 戦略
- 冲突解決

---

### repo-scan

**用途**: コード库扫描

**チェック項目**:
- 依赖漏洞
- コード重复
- 圈复杂度
- ドキュメント完整性

---

## コードレビュー

### click-path-audit

**用途**: 点击路径审计

**使用场景**:
- 用户行为分析
- 转化率最適化
- 漏斗分析

---
