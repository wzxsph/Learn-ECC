# ドキュメントと研究コマンド

## 概要

ドキュメントと研究コマンド用于更新ドキュメント、生成コード地图和研究外部资源。

## コマンド列表

### /update-docs

**用途**: 更新プロジェクトドキュメント

**描述**: 根据コード变更自動更新プロジェクトドキュメント。

**更新内容**:
- README ファイル
- API ドキュメント
- コードコメント
- CHANGELOG

---

### /update-codemaps

**用途**: 重新生成 codemaps

**描述**: 重新生成コード地图，用于コード库导航和理解。

**输出到**: `docs/CODEMAPS/`

---

### /ecc-guide

**用途**: ECC 用户ガイド

**描述**: 表示 ECC プラグイン的完整用户ガイド。

---

## 相关コマンド

- `/update-docs` - 更新ドキュメント
- `/update-codemaps` - 更新コード地图
- `/ecc-guide` - 用户ガイド
