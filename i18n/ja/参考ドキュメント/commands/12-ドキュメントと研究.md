# ドキュメントと研究コマンド

## 概要

ドキュメントと研究コマンド使用更新ドキュメント、生成コード地圖和研究外部資源。

## コマンド一覧

### /update-docs

**用途**: 更新プロジェクトドキュメント

**記述**: 根據コード变更自動更新プロジェクトドキュメント。

**更新內容**:
- README ファイル
- API ドキュメント
- コードコメント
- CHANGELOG

---

### /update-codemaps

**用途**: 重新生成 codemaps

**記述**: 重新生成コード地圖，使用コードベース导航和理解。

**输出到**: `docs/CODEMAPS/`

---

### /ecc-guide

**用途**: ECC ユーザーガイド

**記述**: 表示 ECC プラグイン的完整ユーザーガイド。

---

## 相關コマンド

- `/update-docs` - 更新ドキュメント
- `/update-codemaps` - 更新コード地圖
- `/ecc-guide` - ユーザーガイド
