# プロジェクト管理コマンド

## 概要

プロジェクト管理コマンド用于プロジェクト設定、インフラ設定和ツール链管理。

## コマンド列表

### /projects

**用途**: 列出已知プロジェクト + instinct 統計

**描述**: 列出用户すべて已知プロジェクト及其相关的 instinct 統計情報。

---

### /harness-audit

**用途**: 审计 agent harness 設定

**描述**: 検査和审计 Claude Code 的 harness 設定。

**検査内容**:
- ツール链設定
- プロキシ設定
- MCP サーバー状态
- ルール設定

---

### /model-route

**用途**: ルーティングタスク到合适モデル

**描述**: 根据タスクタイプ选择最合适的モデル（Haiku/Sonnet/Opus）。

**モデル选择ガイド**:
- **Haiku**: 轻量级タスク、コード生成、频繁调用
- **Sonnet**: 主要開発作業、复杂エンコードタスク
- **Opus**: アーキテクチャ决策、深さ推論、研究分析

---

### /pm2

**用途**: PM2 プロセス管理器初期化

**描述**: 初期化 PM2 プロセス管理器設定。

---

### /setup-pm

**用途**: 設定パッケージマネージャー

**描述**: 設定プロジェクト使用的パッケージマネージャー。

**サポート的パッケージマネージャー**:
- npm
- pnpm
- yarn
- bun

**可通過** `CLAUDE_PACKAGE_MANAGER` **环境変数設定**

---

### /project-init

**用途**: プロジェクト初期化

**描述**: 初期化新プロジェクト的標準構造和設定。

---

## 相关コマンド

- `/projects` - 列出プロジェクト
- `/harness-audit` - 审计設定
- `/model-route` - モデルルーティング
