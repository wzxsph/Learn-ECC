| kotlin-build-resolver | Kotlin/Gradle ビルドエラー修正 | sonnet | Read, Write, Edit, Bash, Grep, Glob |

| swift-build-resolver | Swift/Xcode ビルドエラー修正 | sonnet | Read, Write, Edit, Bash, Grep, Glob |

## build-error-resolver

### 名称と用途
TypeScript ビルド和タイプエラー解決专家。修正ビルド失敗或タイプエラー，专注于让ビルド变绿，不做アーキテクチャ修改。

### 能力説明
- TypeScript エラー解決
- ビルドエラー修正
- 依赖問題修正
- 設定エラー解決
- 最小化 diff 修正

### 適用シナリオ
- ビルド失敗时
- TypeScript タイプエラー发生时
- モジュール解析問題时

### 使用ツール一覧
- Read: 読み取りファイル内容
- Write: 書き込み修正
- Edit: 編集ファイル
- Bash: 実行 tsc, npm build, eslint
- Grep: 搜索エラーパターン
- Glob: 検索相关ファイル

### 他の Agent との連携
- 不做リファクタリング → 使用 refactor-cleaner
- 不做アーキテクチャ变更 → 使用 architect
- 不追加新機能 → 使用 planner
- 不修正テスト → 使用 tdd-guide
- 不处理セキュリティ問題 → 使用 security-reviewer

### コア原則
- 只修正エラー，不要リファクタリング
- 最小化变更
- 検証每次修正后ビルド通過

|----------|------|----------|
| undefined: X | 缺少インポート、拼写エラー | 追加インポート或修正サイズ写 |

## java-build-resolver

### 名称と用途
Java/Maven/Gradle ビルド、コンパイル和依赖エラー解決专家。自動检测 Spring Boot 或 Quarkus 并应用フレームワーク特定修正。

### 能力説明
- Java コンパイルエラー診断
- Maven 和 Gradle ビルド設定問題修正
- 依赖冲突和バージョン不匹配解決
- 注解处理器エラー処理（Lombok、MapStruct、Spring、Quarkus）
- Checkstyle 和 SpotBugs 违规修正

### 適用シナリオ
- Java ビルド失敗时
- Maven/Gradle 依赖冲突时
- フレームワーク特定ビルド問題时

### 使用ツール一覧
- Read: 読み取りファイル内容
- Write: 書き込み修正
- Edit: 編集ファイル
- Bash: 実行 ./mvnw compile, ./gradlew build, dependency:tree
- Grep: 搜索エラーパターン
- Glob: 検索相关ファイル

### 他の Agent との連携
- java-reviewer レビューコード品質
- security-reviewer 处理セキュリティ相关問題

### フレームワーク检测
自動检测プロジェクトフレームワーク：
- 如果パッケージ含 `quarkus` → 应用 [QUARKUS] ルール
- 如果パッケージ含 `spring-boot` → 应用 [SPRING] ルール
- 两者都あり → 应用两个ルール集

### 常见エラー修正パターン

| エラータイプ | 原因 | 修正メソッド |
|----------|------|----------|
| cannot find symbol | 缺少インポート、拼写エラー | 追加インポート或依赖 |
| incompatible types | タイプ不匹配 | 追加显式转换 |
| package X does not exist | 缺少依赖 | 追加到 pom.xml/build.gradle |
| No qualifying bean of type X | 缺少 @Component 或コンポーネント扫描 | 追加注解或修正扫描パッケージ |
