# コードレビュー担当 Agent

コードレビュー担当 Agent は、コード品質、セキュリティ、保守性のレビューを専門とします。

## Agent 一覧

| Agent 名 | 用途 | 使用モデル | コアツール |

| code-reviewer | 通用汎用コードレビュー | sonnet | Read, Grep, Glob, Bash |

| go-reviewer | Go コードレビュー（並行処理、エラー処理、パフォーマンス） | sonnet | Read, Grep, Glob, Bash |

| flutter-reviewer | Flutter/Dart コードレビュー（状態管理、パフォーマンス） | sonnet | Read, Grep, Glob, Bash |

## code-reviewer

### 名称と用途
通用汎用コードレビュー，主动レビューコード的品質、セキュリティ和保守性。すべてコード变更后must使用。

### 能力説明
- コード品質検査（関数サイズ、嵌套深さ、コード重复）
- セキュリティ脆弱性検出（SQL インジェクション、XSS、平文凭证）
- React/Next.js パターン検査
- Node.js/バックエンドパターン検査
- パフォーマンス問題识别
- ベストプラクティス提案

### 適用シナリオ
- コード書き込み或修改后
- Pull Request レビュー
- マージ到共享ブランチ前

### 使用ツール一覧
- Read: 読み取りファイル内容
- Grep: 搜索コードパターン
- Glob: 検索ファイル
- Bash: 実行 git diff 和 lint コマンド

### 他の Agent との連携
- 发现セキュリティ問題时 Escalation 到 security-reviewer
- 发现ビルドエラー时使用 build-error-resolver
- 发现必要リファクタリング时使用 refactor-cleaner

## python-reviewer

### 名称と用途
Python コードレビュー专家，专注于 PEP 8 合规、Pythonic 惯用语、型ヒント、セキュリティ和パフォーマンス。

### 能力説明
- PEP 8 风格検査
- Pythonic 惯用语推广（列表推导、枚举、with 语句）
- 型ヒント検証
- SQL インジェクション检测
- コルーチン与非同期パターン検査
- フレームワーク特定検査（Django、FastAPI、Flask）

### 適用シナリオ
- Python プロジェクト中的すべてコード变更
- Django/FastAPI/Flask 特定プロジェクト

### 使用ツール一覧
- Read: 読み取りファイル内容
- Grep: 搜索コードパターン
- Glob: 検索 Python ファイル
- Bash: 実行 mypy, ruff, black, bandit, pytest

### 他の Agent との連携
- 与 security-reviewer 共享セキュリティ検査ルール
- 使用 tdd-guide 确保テストカバレッジ
