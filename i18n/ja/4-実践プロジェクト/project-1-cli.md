# プロジェクト 1: CLI ツール

ECC を使用してコマンドラインツールを構築します。

## プロジェクト目標

**予想時間**: 2-3 時間

`ecc-gen` ツールを作成して、ECC プロジェクトコンポーネント（Agent、Skill、Hook）を生成します。

## 機能要件

### 1. コア機能
- `ecc-gen agent <name>` - Agent ファイルを生成
- `ecc-gen skill <name>` - Skill ファイルを生成
- `ecc-gen hook <name>` - Hook 設定を生成

### 2. インタラクティブ機能
- インタラクティブな Q&A 生成
- テンプレート選択
- 出力パス指定

### 3. 上級機能
- 一括生成
- カスタムテンプレート
- 設定ファイルサポート

## 技術ソリューション

### ディレクトリ構造

```
ecc-gen/
├── src/
│   ├── commands/
│   │   ├── agent.js
│   │   ├── skill.js
│   │   └── hook.js
│   ├── generators/
│   │   └── templates/
│   └── index.js
├── package.json
└── README.md
```

### 使用ライブラリ
- `commander` - CLI パラメータ解析
- `inquirer` - インタラクティブ Q&A
- `ejs` - テンプレートエンジン
