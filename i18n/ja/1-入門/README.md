# 第1段階：入門

本段階は初心者向けに設計されており、ECCへの素早い理解と使用開始を支援します。

## 学習目標

本段階を完了した後、以下のことができます：
- ECCの基本概念と応用シーンを理解する
- Claude Codeのインストールと基本設定が完了する
- 常用コマンド熟練に使用できる
- 独立して単純な開発任務を完了できる

## 予想所要時間

2-3時間

## 前提条件

- コマンドライン基礎
- JavaScript/Node.js 基礎（了解程度で可）
- Git 基礎概念
