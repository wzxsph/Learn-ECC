# Stage 2: コア能力 (Core Skills Mastery)

> すべてのコアコマンドを精通、Agentコラボレーションを掌握、Hooks/Skills/Rulesのカスタマイズが可能

## コース概要

本段階ではClaude Codeのコア能力を深く掌握带你深入，包括75+コマンドシステム、6類Agentコラボレーション、Hooksカスタマイズ、Skills開発、Rules作成。 本段階を完了すると、Claude Codeを完全に制御し、高度にカスタマイズされた開発ワークフローを実現できる。

## 学習時間

**予想時間**: 1週間（毎日2-3時間）

## 前置要件

- [Stage 1: 入門基礎](../1-入門/README.md) を完了
- Claude Code環境が設定済み
- 基本ターミナル操作に慣れている

## 学習目標

本段階を完了した後、以下のことができる：

1. **コマンドシステム精通**: すべての75+コマンドを掌握し、シーンに応じて最適なコマンド組み合わせを選択できる
2. **Agentコラボレーション**: 6類Agentの責務を理解し、組み合わせて複雑なタスクを完了できる
3. **Hooksカスタマイズ**: PreToolUse/PostToolUse/Stop Hooksを作成できる
4. **Skills開発**: カスタムSkillsを作成・管理し、ワークフローパターンを蓄積できる
5. **Rules作成**: プロジェクトレベルのRulesを作成し、コードレビューと品質基準をカスタマイズできる

## 学習提案

1. **循序渐进**: モジュール順に進み、各モジュールを完了してから次へ
2. **手を動かす**: 各演習は実際に実行し、見ただけで動かさない
3. **教材を読む**: 各モジュールには詳細な技術ドキュメントリンクが添付
4. **ノートを取る**: 自分の使用心得とベストプラクティスを記録

## 次の段階

本段階を完了すると、[Stage 3: 上級応用](../3-熟練への道/README.md)に進み、Agentカスケード、自動化ワークフロー、CI/CD統合などの上級テーマを学習します。
