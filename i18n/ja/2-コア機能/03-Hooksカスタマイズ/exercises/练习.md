# Hooks カスタマイズ演習

## 演習目標

Claude Code フックの作成、構成、デバッグをマスターする。

## 演習内容

### 演習 1：PreToolUse フックを作成
フック機能を実装：
- ツール実行前に呼び出しパラメータを記録
- ホワイトリスト/ブラックリストフィルタリングをサポート

### 演習 2：PostToolUse フックを作成
フック機能を実装：
- ツール実行後に出力をフォーマット
- 実行時間を記録

### 演習 3：非同期フックを構成
非同期フックを処理するために作成：
- 外部 API 呼び出し
- ファイルシステム監視

## フックテンプレート

```json
{
  "name": "my-hook",
  "matcher": {
    "tool": "Bash",
    "path": "src/**/*.js"
  },
  "hooks": [
    {
      "type": "PostToolUse",
      "command": "echo 'ツール実行済み: ${tool_name}'"
    }
  ]
}
```

## 検証
1. トリガー条件シーン
2. フック出力をチェック
3. ログ記録を検証
