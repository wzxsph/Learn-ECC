# 演習：コマンドシステム精通

## 演習 1: ベースコマンドを作成

### 目標
`/mycommand` コマンドを作成、以下の機能を実装：
- 現在時刻とプロジェクトパスを出力
- `--verbose` オプションをサポート

### 手順
1. `commands/` ディレクトリに新しい `.md` ファイルを作成
2. YAML frontmatterを追加
3. コマンドロジックを実装
4. テストして検証

---

## 演習 2: パラメータ付きコマンドを実装

### 目標
`/search` コマンドを作成、以下を実装：
- 検索キーワードパラメータを受け取る
- `--limit` パラメータで結果数を制御
- フォーマット化された出力を表示

### 完了基準
- [ ] キーワード検索が動作する
- [ ] limitパラメータが機能する
- [ ] 出力がフォーマット化されている

---

## 演習 3: コマンドデバッグ

### 目標
既存のコマンドをデバッグし、エラー処理とログ出力を追加。

### 手順
1. 既存のコマンドを選択

4. テストして検証
