# 04 Skills 開発演習

## 演習目標

Claude Code Skill ワークフローの作成とデバッグをマスターする。

## 演習内容

### 演習 1：ベース Skill を作成
`my-skill.md` を作成し、以下を含める：
- When to Use 部分
- How It Works 部分
- 少なくとも2つの使用例

### 演習 2：Skill ワークフローを実装
完全な Skill を作成：
- トリガー条件を定義
- 実行ステップを記述
- エラー処理を追加

### 演習 3：Skill デバッグ
既存の Skill をデバッグし、ログと検証ロジックを追加。

## Skill テンプレート

```markdown
# 私の Skill

## When to Use
この Skill をいつ使用するかを説明。

## How It Works
Skill の作動原理と実行フローを説明。

## Examples

### 例 1
最初の使用シーンを説明。

### 例 2
2番目の使用シーンを説明。

## 注意事項
- 注意点 1
- 注意点 2
```

## 検証
1. Skill を呼び出して機能を検証
2. 出力が予想通りかどうかチェック
