# Rules 作成演習

## 演習目標

Claude Code 規則の作成と組織をマスターする。

## 演習内容

### 演習 1：ベース規則を作成
`my-rule.md` を作成し、以下を含める：
- 規則目的の说明
- 適用シーンの定義
- 具体的な規範条項

### 演習 2：マルチレベル規則を実装
分层規則を作成：
- 全般通用規則
- プロジェクト特定規則
- 言語関連規則

### 演習 3：規則デバッグ
規則優先度と覆盖ロジックをデバッグ。

## Rule テンプレート

```markdown
# 私の規則

## 目的
規則が解決する問題を説明。

## 適用範囲
規則が適用されるシーンまたはファイルタイプを説明。

## 規範
- 規範 1
- 規範 2

## 検証方式
規則が遵守されているかどうかを検証する方法を説明。
```

## 検証
1. 規則シーンをトリガー
2. 規則が有効かどうかをチェック
3. 優先度の覆盖を検証
