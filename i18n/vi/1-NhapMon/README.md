# Giai đoạn 1: Nhập môn

Giai đoạn này được thiết kế cho người hoàn toàn mới, giúp bạn nhanh chóng hiểu ECC và bắt đầu sử dụng.

## Mục tiêu học tập

Sau khi hoàn thành giai đoạn này, bạn sẽ có khả năng:
- Hiểu các khái niệm cơ bản của ECC và các kịch bản ứng dụng
- Hoàn thành cài đặt và cấu hình Claude Code
- Sử dụng thành thạo các lệnh thường dùng
- Độc lập hoàn thành các nhiệm vụ phát triển đơn giản

## Thời gian ước tính

2-3 giờ

## Yêu cầu tiên quyết

- Cơ bản về command line
- Cơ bản về JavaScript/Node.js (chỉ cần hiểu là được)
- Khái niệm cơ bản về Git
