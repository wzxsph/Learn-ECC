# Bài tập: Phát triển Skills

## Mục tiêu bài tập

Nắm vững việc tạo và debug Claude Code Skill workflow.

## Nội dung bài tập

### Bài tập 1: Tạo Skill cơ bản
Tạo `my-skill.md`, bao gồm:
- Phần When to Use
- Phần How It Works
- ít nhất 2 ví dụ sử dụng

### Bài tập 2: Implement workflow Skill
Tạo Skill hoàn chỉnh:
- Định nghĩa điều kiện trigger
- Viết các bước thực thi
- Thêm xử lý lỗi

### Bài tập 3: Debug Skill
Debug Skill hiện có, thêm log và logic xác minh.

## Skill template

```markdown
# Skill của tôi

## When to Use
Mô tả khi nào sử dụng Skill này.

## How It Works
Mô tả nguyên lý hoạt động và quy trình thực thi của Skill.

## Examples

### Ví dụ 1
Mô tả kịch bản sử dụng đầu tiên.

### Ví dụ 2
Mô tả kịch bản sử dụng thứ hai.

## Lưu ý
- Điểm cần chú ý 1
- Điểm cần chú ý 2
```

## Xác minh
1. Gọi Skill xác minh chức năng
2. Kiểm tra output có đúng như kỳ vọng không
