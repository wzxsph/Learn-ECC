# Bài tập: Agent协作

## Mục tiêu bài tập

Học cách điều phối nhiều Agent trong Claude Code để hoàn thành nhiệm vụ phức tạp.

## Nội dung bài tập

### Bài tập 1: Thực thi nhiệm vụ song song
Khởi động 2 Agent song song, lần lượt hoàn thành:
- Agent A: Phân tích cấu trúc dự án
- Agent B: Kiểm tra quy tắc code

### Bài tập 2: Chuỗi nhiệm vụ
Implement chuỗi gọi 3 Agent:
- Agent 1: Thu thập yêu cầu
- Agent 2: Generate code
- Agent 3: Review code

### Bài tập 3: Tổng hợp kết quả
Gộp output của nhiều Agent thành báo cáo thống nhất.

## Template

```markdown
# Phương án Agent协作

## Mô tả nhiệm vụ
Mô tả ngắn gọn nhiệm vụ协作 cần hoàn thành.

## Agent tham gia
| Agent | Vai trò | Nhiệm vụ |
|-------|------|------|
| agent-a | Phân tích | xxx |
| agent-b | Thực thi | xxx |

## Quy trình thực hiện
1. Khởi động Agent A
2. Khởi động Agent B
3. Tổng hợp kết quả

## Output kỳ vọng
Mô tả format output cuối cùng.
```

## Xác minh

Kiểm tra output của các Agent có được tổng hợp đúng không.
