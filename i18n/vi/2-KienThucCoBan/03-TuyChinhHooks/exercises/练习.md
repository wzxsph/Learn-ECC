# Bài tập: Tùy chỉnh Hooks

## Mục tiêu bài tập

Nắm vững việc tạo, cấu hình và debug hooks của Claude Code.

## Nội dung bài tập

### Bài tập 1: Tạo PreToolUse Hook
Implement chức năng hook:
- Ghi log tham số gọi trước khi công cụ thực thi
- Hỗ trợ lọc whitelist/blacklist

### Bài tập 2: Tạo PostToolUse Hook
Implement chức năng hook:
- Format output sau khi công cụ thực thi
- Ghi log thời gian thực thi

### Bài tập 3: Cấu hình async hook
Tạo async hook xử lý:
- Gọi API bên ngoài
- Giám sát filesystem

## Template hook

```json
{
  "name": "my-hook",
  "matcher": {
    "tool": "Bash",
    "path": "src/**/*.js"
  },
  "hooks": [
    {
      "type": "PostToolUse",
      "command": "echo 'Công cụ đã thực thi: ${tool_name}'"
    }
  ]
}
```

## Xác minh
1. Kích hoạt kịch bản điều kiện
2. Kiểm tra output của hook
3. Xác minh ghi log
