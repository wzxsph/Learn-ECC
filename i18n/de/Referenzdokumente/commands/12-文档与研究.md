# 文档与研究命令

## 概述

文档与研究命令用于更新文档、生成代码地图和研究外部资源。

## 命令列表

### /update-docs

**用途**: 更新项目文档

**描述**: 根据代码变更自动更新项目文档。

**更新内容**:
- README 文件
- API 文档
- 代码注释
- CHANGELOG

---

### /update-codemaps

**用途**: 重新生成 codemaps

**描述**: 重新生成代码地图，用于代码库导航和理解。

**输出到**: `docs/CODEMAPS/`

---

### /ecc-guide

**用途**: ECC 用户指南

**描述**: 显示 ECC 插件的完整用户指南。

---

## 相关命令

- `/update-docs` - 更新文档
- `/update-codemaps` - 更新代码地图
- `/ecc-guide` - 用户指南
