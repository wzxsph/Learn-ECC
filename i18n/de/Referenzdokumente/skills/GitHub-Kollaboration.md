# GitHub 与协作技能

## 概述

GitHub 与协作技能用于 GitHub 操作、代码审查、PR 管理和团队协作。

## 核心技能

### github-ops

**用途**: GitHub 操作与自动化

**功能**:
- Issue 管理
- PR 创建与审查
- 代码审查自动化
- GitHub Actions 工作流

---

### git-workflow

**用途**: Git 工作流规范

**核心概念**:
- 分支策略
- Commit 规范
- Merge 策略
- 冲突解决

---

### repo-scan

**用途**: 代码库扫描

**检查项**:
- 依赖漏洞
- 代码重复
- 圈复杂度
- 文档完整性

---

## 代码审查

### click-path-audit

**用途**: 点击路径审计

**使用场景**:
- 用户行为分析
- 转化率优化
- 漏斗分析

---

## 相关技能

- `github-ops` - GitHub 操作
- `git-workflow` - Git 工作流
- `repo-scan` - 代码库扫描
- `click-path-audit` - 点击路径审计
