# 后端与 API 技能

## 概述

后端与 API 相关技能用于构建服务器端应用、API 设计和数据库操作。

## 核心技能

### backend-patterns

**用途**: 后端开发最佳实践

**核心概念**:
- 分层架构（Controller → Service → Repository）
- 依赖注入
- 错误处理策略
- 缓存模式
- 消息队列集成

---

### api-design

**用途**: API 设计与RESTful 规范

**核心概念**:
- RESTful 路由设计
- 请求/响应格式
- 认证与授权
- 版本控制策略
- 限流与缓存

---

### api-connector-builder

**用途**: API 连接器构建工具

**使用场景**:
- 第三方 API 集成
- Webhook 处理
- OAuth 流程实现
- API 错误重试机制

---

## 数据库技能

### database-migrations

**用途**: 数据库迁移管理

**核心概念**:
- 迁移文件组织
- 数据回滚策略
- 种子数据管理
- 零停机部署

---

### mysql-patterns

**用途**: MySQL 数据库最佳实践

**核心概念**:
- 索引优化
- 查询优化
- 分表策略
- 主从复制

---

### postgres-patterns

**用途**: PostgreSQL 数据库最佳实践

**核心概念**:
- JSONB 操作
- 全文搜索
- 分区表
- 高级索引策略

---

### redis-patterns

**用途**: Redis 缓存模式

**核心概念**:
- 缓存策略（Cache-Aside、Write-Through）
- 分布式锁
- 会话存储
- 消息发布/订阅

---

### clickhouse-io

**用途**: ClickHouse 列式存储

**使用场景**:
- 分析型查询
- 日志存储
- 实时分析
- 大数据管道

---

## 消息队列

### logistics-exception-management

**用途**: 物流异常管理

**使用场景**:
- 订单异常处理
- 退货流程
- 库存同步

---
