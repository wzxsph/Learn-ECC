# Phase 2: Kernfähigkeiten (Core Skills Mastery)

> Alle Kernbefehle beherrschen, Agent-Zusammenarbeit meistern, Hooks/Skills/Rules anpassen können

## Kursübersicht

Diese Phase führt dich tief in die Kernfähigkeiten von Claude Code ein, einschließlich 75+ Befehlssystem, 6 Kategorien Agent-Zusammenarbeit, Hooks-Anpassung, Skills-Entwicklung und Rules-Erstellung. Nach Abschluss dieser Phase wirst du Claude Code vollständig beherrschen und hochgradig angepasste Entwicklungsworkflows realisieren können.

## Lernzeit

**Geschätzte Zeit**: 1 Woche (täglich 2-3 Stunden)

## Voraussetzungen

- [Phase 1: Einsteiger](../1-Einstieg/README.md) abgeschlossen
- Claude Code-Umgebung konfiguriert
- Grundlegende Terminal-Operationen vertraut

## Lernziele

Nach Abschluss dieser Phase wirst du in der Lage sein:

1. **Befehlssystem meistern**: Alle 75+ Befehle beherrschen, beste Befehlskombinationen nach Szenario auswählen können
2. **Agent-Zusammenarbeit**: Verstehen was 6 Kategorien von Agents können, komplexe Aufgaben durch Kombination meistern
3. **Hooks anpassen**: PreToolUse/PostToolUse/Stop Hooks schreiben können
4. **Skills entwickeln**: Eigene Skills erstellen und verwalten, Workflow-Muster konsolidieren
5. **Rules schreiben**: Projekt-level Regeln schreiben, Code-Review und Qualitätsstandards anpassen

## Lernempfehlungen

1. **Schritt für Schritt**: Module sequentiell lernen, jedes Modul abschließen bevor du zum nächsten gehst
2. **Praktisch anwenden**: Jede Übung tatsächlich ausführen, nicht nur lesen
3. **Material lesen**: Jedes Modul hat detaillierte technische Dokumentationslinks
4. **Notizen machen**: Deine eigenen Nutzungstipps und Best Practices notieren
