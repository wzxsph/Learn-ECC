# Rules编写练习

## 练习目标

掌握 Claude Code 规则的创建和组织。

## 练习内容

### 练习 1：创建基础规则
创建 `my-rule.md`，包含：
- 规则目的说明
- 适用场景定义
- 具体规范条款

### 练习 2：实现多级规则
创建分层规则：
- 全局通用规则
- 项目特定规则
- 语言相关规则

### 练习 3：规则调试
调试规则优先级和覆盖逻辑。

## Rule 模板

```markdown
# 我的规则

## 目的
描述规则要解决什么问题。

## 适用范围
描述规则适用的场景或文件类型。

## 规范
- 规范 1
- 规范 2

## 验证方式
描述如何验证规则是否被遵守。
```

## 验证
1. 触发规则场景
2. 检查规则是否生效
3. 验证优先级覆盖
