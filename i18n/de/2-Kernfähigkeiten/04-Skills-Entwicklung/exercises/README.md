# Skills开发练习

## 练习目标

学会创建和调试 Claude Code Skill 工作流。

## 练习内容

### 练习 1：创建基础 Skill
创建 `my-skill.md`，包含：
- When to Use 部分
- How It Works 部分
- 至少 2 个使用示例

### 练习 2：实现 Skill 工作流
创建完整 Skill：
- 定义触发条件
- 编写执行步骤
- 添加错误处理

### 练习 3：Skill 调试
调试现有 Skill，添加日志和验证逻辑。

## Skill 模板

```markdown
# 我的 Skill

## When to Use
描述何时使用此 Skill。

## How It Works
描述 Skill 的工作原理和执行流程。

## Examples

### 示例 1
描述第一个使用场景。

### 示例 2
描述第二个使用场景。

## 注意事项
- 注意点 1
- 注意点 2
```

## 验证
1. 调用 Skill 验证功能
2. 检查输出是否符合预期
