# Befehlssystem-Übungen

## Übungsziele

Claude Code-Befehlssystem erstellen, konfigurieren und debuggen beherrschen.

## Übungsinhalt

### Übung 1: Basisbefehl erstellen
`/mycommand` Befehl erstellen, mit folgenden Funktionen:
- Aktuelle Zeit und Projektpfad ausgeben
- `--verbose` Option unterstützen

### Übung 2: Parameterbefehl implementieren
`/search` Befehl erstellen, mit:
- Suchbegriff als Parameter akzeptieren
- `--limit` Parameter zur Kontrolle der Ergebnisanzahl
- Formatierte Ausgabe anzeigen

### Übung 3: Befehl debuggen
Bestehenden Befehl debuggen, Fehlerbehandlung und Log-Ausgabe hinzufügen.

## Vorlage

```markdown
# Mein Befehl

## Verwendung
Befehlshauptfunktion beschreiben.

## Nutzung
`/mycommand [Optionen] <Parameter>`

## Optionen
- `--option`: Optionsbeschreibung

## Beispiel
`/mycommand --option wert`
```

## Verifizierung
Nach Abschluss der Übung Befehl ausführen um zu verifizieren, dass Funktion normal.
