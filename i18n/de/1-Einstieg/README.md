# Phase 1: Einsteiger

Diese Phase ist für Benutzer ohne Vorkenntnisse konzipiert und hilft dir, ECC schnell zu verstehen und zu nutzen.

## Lernziele

Nach dieser Phase wirst du in der Lage sein:
- Die grundlegenden Konzepte und Anwendungsfälle von ECC zu verstehen
- Die Installation und Grundkonfiguration von Claude Code abzuschließen
- Häufig verwendete Befehle sicher anzuwenden
- Einfache Entwicklungsaufgaben selbstständig zu erledigen

## Geschätzte Dauer

2-3 Stunden

## Voraussetzungen

- Grundlagen der Kommandozeile
- Grundlagen von JavaScript/Node.js (Verständnis reicht)
- Git-Grundkonzepte
