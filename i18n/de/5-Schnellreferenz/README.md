# Phase 5: Schnellreferenz

Dieses Cheatsheet bietet eine Kurzreferenz für häufig verwendete Befehle, Agents und Arbeitsabläufe.

## Inhalt

| Cheatsheet | Inhalt |
|------------|--------|
| [Befehls-Cheatsheet](./cheatsheet-commands.md) | Slash-Commands, Skriptbefehle, Parameterbeschreibungen |
| [Agent-Cheatsheet](./cheatsheet-agents.md) | Liste der integrierten Agents und ihrer Fähigkeiten |
| [Arbeitsablauf-Cheatsheet](./cheatsheet-workflows.md) | Häufig verwendete Arbeitsablauf-Vorlagen |

## Verwendungsempfehlungen

- Speichere häufig verwendete Befehle in deinen Notizen
- Drucke die Agent-Liste als Referenz aus
- Speichere Arbeitsablauf-Vorlagen in deinen Favoriten
