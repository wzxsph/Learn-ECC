# ขั้นตอนที่ 4: โครงการปฏิบัติ

สร้างความเข้าใจที่เรียนผ่านด้วยโครงการจริง

## วัตถุประสงค์การเรียนรู้

เมื่อเรียนจบขั้นตอนนี้ คุณจะสามารถ:
- สร้างเครื่องมือ CLI ที่สมบูรณ์จากศูนย์
- ดำเนินการ quy trình review code อัตโนมัติ
- ออกแบบ workflow อัตโนมัติ end-to-end
- สร้างแพลตฟอร์มการทำงานร่วมกันแบบหลาย Agent

## เวลาโดยประมาณ

10-15 ชั่วโมง

## ข้อกำหนดเบื้องต้น

- เรียนจบ [ขั้นตอนที่ 2: ความสามารถหลัก](../2-ความสามารถหลัก/README.md)
- เชี่ยวชาญอย่างน้อยหนึ่งภาษา script
- มีประสบการณ์พัฒนาโครงการ

## คำแนะนำ

เลือกโครงการที่สนใจเพื่อเจาะลึกทำให้เสร็จ แทนที่จะทำทุกโครงการอย่างผิวเผิน
แนะนำทำตามลำดับ: เครื่องมือ CLI → Review code → Workflow อัตโนมัติ → ระบบ Agent (ความยากเพิ่มขึ้นทีละขั้น)
