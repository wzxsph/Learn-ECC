## เรื่องราวพื้นหลัง

### ทำไมจึงมีโครงการนี้?

ฉันไม่เก่งภาษาอังกฤษ การอ่านเอกสาร ECC เป็นภาษาอังกฤษค่อนข้างลำบาก ดังนั้นฉันจึงแปลคำสั่ง (Commands) ตัวแทน (Agents) ทักษะ (Skills) ใน ECC เป็นภาษาจีน และจัดเป็นหลักสูตรการเรียนรู้นี้

เอกสารอ้างอิงมาจากการแปลเอกสารของโครงการ ECC ต้นฉบับเป็นภาษาจีน - บอกคุณว่าแต่ละคำสั่ง แต่ละ Agent ทำอะไร

### หลักสูตรนี้มาจากไหน?

ตรงไปตรงมา หลักสูตรนี้ถูกสร้างโดย AI AI สร้างเนื้อหาพื้นฐานจากโครงสร้างไฟล์ของ ECC แต่ขาดการคิดอย่างละเอียดและการประเมินจริง หากคุณพบว่าเนื้อหามีปัญหาหรือไม่ชัดเจน ยินดีรับข้อเสนอแนะ

## แนวคิดหลัก

### ECC คืออะไร?

**ECC** (Everything Claude Code) เป็นโครงการปลั๊กอิน Claude Code ([repo ต้นฉบับ](https://github.com/affaan-m/ECC)) มี:
- **75 คำสั่ง** - คำสั่ง slash (เช่น `/plan`, `/code-review`) และคำสั่ง script
- **60 Agent เฉพาะทาง** - code review แก้ไข build ออกแบบ architecture และอื่นๆ
- **232+ Skills** - แม่แบบ workflow ที่ใช้ซ้ำได้
- **ระบบ Hooks** - กลไกอัตโนมัติที่ขับเคลื่อนด้วยเหตุการณ์
- **กฎ Rules** - กฎเกณฑ์การพัฒนาที่เป็นภาคบังคับ

### Learn-ECC คืออะไร?

**Learn-ECC** คือเส้นทางการเรียนรู้ที่เป็นระบบที่ฉันจัดเพื่อเรียน ECC ไม่ใช่ตัว ECC เอง

มันใช้ปรัชญา "เรียนรู้โดยการลงมือทำ" แต่ละขั้นตอนมีทั้งการเรียนทฤษฎีและการฝึกปฏิบัติ โดยการทำโครงการจริง คุณจะเชี่ยวชาญความสามารถหลักของ Claude Code

### เอกสารอ้างอิงคืออะไร?

**เอกสารอ้างอิง** (./เอกสารอ้างอิง/) คือการแปลภาษาจีนของโครงการ ECC ต้นฉบับ ฉันแปลเอกสารภาษาอังกฤษของ ECC เป็นภาษาจีนเพื่อความสะดวกในการค้นหาของฉัน (และคุณ)

เนื้อหาประกอบด้วย:
- คำอธิบายคำสั่งทั้งหมด (แต่ละคำสั่งทำอะไร)
- บทนำโดยละเอียดของ Agent ทั้งหมด
- โครงสร้างไดเรกทอรี Skills และกฎการเขียน
- เอกสารฉบับเต็มของระบบ Hooks
- Best practices และ design patterns

## วิธีใช้หลักสูตรนี้

### ลำดับการเรียนที่แนะนำ

1. **เรียนตามลำดับขั้นตอน** - แต่ละขั้นตอนสร้างบนพื้นฐานของขั้นตอนก่อนหน้า
2. **ทฤษฎี + ปฏิบัติ** - แต่ละบทมีแบบฝึกหัด เข้าใจก่อนแล้วลงมือทำ
3. **ทำแบบฝึกหัดทั้งหมด** - แบบฝึกหัดคือส่วนสำคัญของเส้นทางการเรียน

### คำแนะนำการเรียน

- **ขั้นตอนเริ่มต้น**: แนะนำ 2-3 ชั่วโมง มุ่งเน้นทำความคุ้นเคยกับแนวคิดพื้นฐานและการทำงาน
- **ความสามารถหลัก**: แนะนำ 8-12 ชั่วโมง ทุกโมดูลต้องลงมือทำจริง
- **เส้นทางผู้เชี่ยวชาญ**: แนะนำ 6-8 ชั่วโมง ต้องมีประสบการณ์โครงการบ้าง
- **โครงการปฏิบัติ**: แนะนำ 10-15 ชั่วโมง เลือกโครงการที่สนใจเพื่อเจาะลึก

### จุดตรวจสอบการเรียน

เมื่อเสร็จแต่ละขั้นตอน ยืนยันว่าคุณเชี่ยวชาญแล้ว:
- สามารถทำการทำงานทั้งหมดของขั้นตอนนั้นได้ด้วยตัวเอง
- สามารถอธิบายแนวคิดหลักให้ผู้อื่นเข้าใจ
- สามารถนำไปประยุกต์ใช้กับสถานการณ์อื่นได้

## เคล็ดลับความสำเร็จ

1. **ค่อยเป็นค่อยไป**: อย่าข้ามขั้นตอน แต่ละ Stage คือพื้นฐานของ Stage ถัดไป
2. **เรียนโดยการลงมือทำ**: เรียนแต่ละแนวคิดแล้วฝึกทันที อย่าดูอย่างทำ
3. **ทำแบบฝึกหัดให้เสร็จ**: แต่ละ Stage มีแบบฝึกหัด ทำอย่างจริงจังเพื่อสร้างความเข้าใจ
4. **ใช้ตารางอ้างอิง**: ในการใช้งานประจำวัน ใช้ตารางอ้างอิง อย่าท่องจำ
5. **ขับเคลื่อนด้วยโครงการ**: Stage 4 ใช้โครงการจริงเพื่อเชื่อมทุกความรู้
6. **จดบันทึก**: บันทึกการเรียนรู้และปัญหาที่พบ
7. **สอนผู้อื่น**: สอนผู้อื่นได้ถึงจะเข้าใจจริง

## คำถามที่พบบ่อย (FAQ)

### ถาม: ต้องมีพื้นฐานการเขียนโปรแกรมไหม?

ตอบ: ต้องมีพื้นฐาน command line และการเขียนโปรแกรม (พื้นฐาน JavaScript/Node.js ช่วยเข้าใจฟังก์ชันบางอย่าง) แต่ไม่ต้องเชี่ยวชาญภาษาใดภาษาหนึ่ง ขั้นตอนเริ่มต้นออกแบบสำหรับผู้เริ่มต้นทั้งหมด

### ถาม: ข้ามขั้นตอนได้ไหม?

ตอบ: ขั้นตอน 1-3 แนะนำเรียนตามลำดับ Stage 4 สามารถเลือกโครงการที่สนใจตามความต้องการ Stage 5 ค้นหาได้ตลอดเวลา

### ถาม: หลังเรียนเสร็จทำอะไรได้?

ตอบ: เชี่ยวชาญ production Claude Code workflow พัฒนา review test และอัตโนมัติ code ได้อย่างมีประสิทธิภาพ; ออกแบบและพัฒนาระบบ Agent กำหนดเองได้; ปรับแต่ง Hooks และ Skills ได้

### ถาม: ถ้าเจอปัญหาระหว่างเรียน?

ตอบ: 1) ดู [เอกสารอ้างอิง](./เอกสารอ้างอิง/) บทที่เกี่ยวข้อง; 2) ดู [ตารางคำสั่ง](./5-อ้างอิงเร็ว/cheatsheet-commands.md); 3) ส่ง Issue เพื่อขอความช่วยเหลือ

### ถาม: จะประเมินผลการเรียนอย่างไร?

ตอบ: ใช้ [progress-tracker.md](./progress-tracker.md) ติดตามความก้าวหน้า แต่ละขั้นตอนมีเป้าหมายการเรียนชัดเจน ทำแบบฝึกหัดทั้งหมดคือเชี่ยวชาญเนื้อหาขั้นตอนนั้น

### ถาม: เนื้อหาหลักสูตรจะอัปเดตไหม?

ตอบ: จะอัปเดตอย่างต่อเนื่อง ยินดีส่ง PR มีส่วนร่วม ดูรายละเอียดที่ [contributing.md](./contributing.md)
