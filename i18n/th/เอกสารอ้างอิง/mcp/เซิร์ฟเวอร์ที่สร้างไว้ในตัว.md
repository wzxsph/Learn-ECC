# เซิร์ฟเวอร์ MCP ที่สร้างไว้ในตัว

โครงการ ECC มีการตั้งค่าเซิร์ฟเวอร์ MCP ที่ใช้บ่อยที่สุดที่กำหนดค่าไว้ล่วงหน้าใน `mcp-configs/mcp-servers.json`

## จัดกลุ่มตามฟังก์ชัน

### เครื่องมือพัฒนา

| เซิร์ฟเวอร์ | ประเภท | คำอธิบาย |
|--------|------|------|
| `github` | process | การดำเนินการ GitHub - PRs, issues, repos |
| `jira` | process | การติดตามปัญหา Jira - ค้นหา, สร้าง, อัปเดต, แสดงความคิดเห็น, เปลี่ยนสถานะปัญหา |
| `confluence` | process | การบูรณาการ Confluence Cloud - ค้นหาหน้า, รับเนื้อหา, สำรวจ spaces |
| `filesystem` | process | การดำเนินการระบบไฟล์ (ต้องกำหนดค่า path) |
| `memory` | process | หน่วยความจำถาวรข้ามเซสชัน |
| `omega-memory` | process | หน่วยความจำตัวแทนถาวรแบบเพิ่มประสิทธิภาพ รองรับการค้นหาเชิงความหมาย, การทำงานร่วมกันหลายตัวแทน และกราฟความรู้ |
| `sequential-thinking` | process | การให้เหตุผลแบบลูกโซ่ความคิด |

## การเปิดใช้งานอย่างรวดเร็ว

คัดลอกการตั้งค่าเซิร์ฟเวอร์ที่ต้องการไปยังส่วน `mcpServers` ของ `~/.claude.json` แทนที่ `YOUR_*_HERE` placeholders ด้วยค่าจริง

**ตัวอย่าง - เปิดใช้งานเซิร์ฟเวอร์ GitHub:**

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_your_actual_token"
      },
      "description": "GitHub operations - PRs, issues, repos"
    }
  }
}
```

## ตัวแปรสิ่งแวดล้อมที่ต้องการ

| เซิร์ฟเวอร์ | ตัวแปรสิ่งแวดล้อมที่ต้องการ |
|--------|----------------|
| `github` | `GITHUB_PERSONAL_ACCESS_TOKEN` |
| `jira` | `JIRA_URL`, `JIRA_EMAIL`, `JIRA_API_TOKEN` |
| `firecrawl` | `FIRECRAWL_API_KEY` |
| `supabase` | พารามิเตอร์ `--project-ref` |
| `exa-web-search` | `EXA_API_KEY` |
| `fal-ai` | `FAL_KEY` |
| `browserbase` | `BROWSERBASE_API_KEY` |
| `browser-use` | header คำขอ `x-browser-use-api-key` |
| `confluence` | `CONFLUENCE_BASE_URL`, `CONFLUENCE_EMAIL`, `CONFLUENCE_API_TOKEN` |
| `evalview` | `OPENAI_API_KEY` (ไม่บังคับ) |
