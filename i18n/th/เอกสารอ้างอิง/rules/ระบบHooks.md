# ระบบ Hooks

## ประเภทของ Hook

- **PreToolUse**: ก่อนการรันเครื่องมือ (การตรวจสอบ, การแก้ไขพารามิเตอร์)
- **PostToolUse**: หลังการรันเครื่องมือ (การจัดรูปแบบอัตโนมัติ, การตรวจสอบ)
- **Stop**: เมื่อเซสชันจบ (การตรวจสอบขั้นสุดท้าย)

## การยอมรับอัตโนมัติของสิทธิ์

ใช้ด้วยความระมัดระวัง:
- เปิดใช้สำหรับ plans ที่เชื่อถือได้และกำหนดไว้ชัดเจน
- ปิดใช้สำหรับงานสำรวจ
- ห้ามใช้ dangerously-skip-permissions flag อย่างเด็ดขาด
- กำหนดค่า `allowedTools` ใน `~/.claude.json` แทน

## แนวปฏิบัติที่ดีที่สุดของ TodoWrite

ใช้เครื่องมือ TodoWrite เพื่อ:
- ติดตามความคืบหน้าของงานหลายขั้นตอน
- ยืนยันความเข้าใจของคำสั่ง
- เปิดใช้การชี้นำแบบเรียลไทม์
- แสดงขั้นตอนการ implementation ระดับละเอียด

รายการ Todo เปิดเผย:
- ขั้นตอนที่ลำดับไม่ถูกต้อง
- รายการที่ขาดหาย
- รายการที่ไม่จำเป็นเกินไป
- Granularity ที่ผิด
- ความเข้าใจผิดของความต้องการ
