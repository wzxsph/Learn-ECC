# ตัวแทนตรวจสอบโค้ด

ตัวแทนประเภทตรวจสอบโค้ดมีหน้าที่ตรวจสอบคุณภาพ ความปลอดภัย และการบำรุงรักษาของโค้ดโดยเฉพาะ

## รายการตัวแทน

| ชื่อตัวแทน | การใช้งาน | โมเดลที่ใช้ | เครื่องมือหลัก |
|------------|------|----------|----------|
| code-reviewer | ผู้เชี่ยวชาญตรวจสอบโค้ดทั่วไป | sonnet | Read, Grep, Glob, Bash |
| python-reviewer | การตรวจสอบโค้ด Python (PEP 8, Type hints, ความปลอดภัย) | sonnet | Read, Grep, Glob, Bash |
| go-reviewer | การตรวจสอบโค้ด Go (Concurrency, การจัดการข้อผิดพลาด, ประสิทธิภาพ) | sonnet | Read, Grep, Glob, Bash |
| kotlin-reviewer | การตรวจสอบโค้ด Kotlin/Android (Coroutines, Compose) | sonnet | Read, Grep, Glob, Bash |
| rust-reviewer | การตรวจสอบโค้ด Rust (Ownership, Lifetime ความปลอดภัย) | sonnet | Read, Grep, Glob, Bash |
| cpp-reviewer | การตรวจสอบโค้ด C++ (ความปลอดภัยหน่วยความจำ, Modern C++ idioms) | sonnet | Read, Grep, Glob, Bash |

| swift-reviewer | การตรวจสอบโค้ด Swift (Protocols, Concurrency, ARC memory management) | sonnet | Read, Grep, Glob, Bash |

## code-reviewer

### ชื่อและการใช้งาน
ผู้เชี่ยวชาญตรวจสอบโค้ดทั่วไป ตรวจสอบคุณภาพ ความปลอดภัย และการบำรุงรักษาของโค้ดอย่างแข็งขัน ต้องใช้หลังจากทุกการเปลี่ยนแปลงโค้ด

### ความสามารถ
- การตรวจสอบคุณภาพโค้ด (ขนาดฟังก์ชัน ความลึกของการซ้อน การซ้ำกันของโค้ด)
- การตรวจจับช่องโหว่ด้านความปลอดภัย (SQL injection, XSS, Hardcoded credentials)
- การตรวจสอบรูปแบบ React/Next.js
- การตรวจสอบรูปแบบ Node.js/Backend
- การระบุปัญหาประสิทธิภาพ
- คำแนะนำแนวปฏิบัติที่ดีที่สุด

### สถานการณ์การใช้งาน
- หลังจากเขียนหรือแก้ไขโค้ด
- การตรวจสอบ Pull Request
- ก่อนรวมเข้ากับ branch ที่ใช้ร่วมกัน

### รายการเครื่องมือที่ใช้
- Read: อ่านเนื้อหาไฟล์
- Grep: ค้นหารูปแบบโค้ด
- Glob: ค้นหาไฟล์
- Bash: รันคำสั่ง git diff และ lint

### การทำงานร่วมกับตัวแทนอื่น
- เมื่อพบปัญหาความปลอดภัย ให้ Escalation ไปยัง security-reviewer
- เมื่อพบข้อผิดพลาดในการ build ใช้ build-error-resolver
- เมื่อพบว่าต้อง refactor ใช้ refactor-cleaner

## python-reviewer

### ความสามารถ

- การตรวจสอบ Type hints
- การตรวจจับ SQL injection
- การตรวจสอบ Coroutine และ Async patterns
- การตรวจสอบเฉพาะ Framework (Django, FastAPI, Flask)

### สถานการณ์การใช้งาน
- ทุกการเปลี่ยนแปลงโค้ดในโปรเจกต์ Python
- โปรเจกต์ Django/FastAPI/Flask เฉพาะ

### รายการเครื่องมือที่ใช้
- Read: อ่านเนื้อหาไฟล์
- Grep: ค้นหารูปแบบโค้ด
- Glob: ค้นหาไฟล์ Python
- Bash: รัน mypy, ruff, black, bandit, pytest

### การทำงานร่วมกับตัวแทนอื่น
- แบ่งปันกฎตรวจสอบความปลอดภัยกับ security-reviewer
