# คำสั่งระบบ Hookify

## ภาพรวม

ระบบ Hookify ใช้สำหรับสร้างและจัดการ hooks ป้องกันพฤติกรรมที่ไม่พึงประสงค์และอัตโนมัติ workflow

## รายการคำสั่ง

### /hookify

**วัตถุประสงค์**: สร้าง hooks เพื่อป้องกันพฤติกรรมที่ไม่พึงประสงค์

**คำอธิบาย**: สร้าง hooks แบบกำหนดเองเพื่อป้องกันพฤติกรรมที่ไม่ต้องการ

**สถานการณ์การใช้งาน**:
- ป้องกันการลบโดยไม่ตั้งใจ
- ป้องกันคำสั่งที่เป็นอันตราย
- การจัดรูปแบบอัตโนมัติ
- การตรวจสอบคุณภาพ

**ประเภท Hook**:
- **PreToolUse**: ก่อนการรันเครื่องมือ
- **PostToolUse**: หลังการรันเครื่องมือ
- **Stop**: เมื่อเซสชันจบ

---

### /hookify-list

**วัตถุประสงค์**: แสดงรายการกฎ hookify ทั้งหมดที่ตั้งค่าไว้

**คำอธิบาย**: แสดงกฎ hookify ทั้งหมดที่ตั้งค่าในปัจจุบัน

---

### /hookify-configure

**วัตถุประสงค์**: เปิด/ปิดกฎ hookify แบบโต้ตอบ

**คำอธิบาย**: การตั้งค่าแบบโต้ตอบสำหรับการเปิดและปิดสถานะกฎ hookify

---

### /hookify-help

**วัตถุประสงค์**: ความช่วยเหลือระบบ Hookify

**คำอธิบาย**: รับข้อมูลความช่วยเหลือโดยละเอียดเกี่ยวกับระบบ Hookify

---

## ตัวอย่างการตั้งค่า Hook

```json
{
  "matcher": { "pattern": "Bash|Edit|Write" },
  "hooks": [
    {
      "type": "command",
      "command": "echo 'Running quality check'"
    }
  ]
}
```

---

## คำสั่งที่เกี่ยวข้อง

- `/hookify` - สร้าง hook
- `/hookify-list` - แสดงรายการกฎ
- `/hookify-configure` - ตั้งค่ากฎ
- `/hookify-help` - รับความช่วยเหลือ
