# คำสั่งเอกสารและการวิจัย

## ภาพรวม

คำสั่งเอกสารและการวิจัยใช้สำหรับอัปเดตเอกสาร สร้าง codemaps และวิจัยทรัพยากรภายนอก

## รายการคำสั่ง

### /update-docs

**วัตถุประสงค์**: อัปเดตเอกสารโครงการ

**คำอธิบาย**: อัปเดตเอกสารโครงการโดยอัตโนมัติตามการเปลี่ยนแปลงโค้ด

**เนื้อหาที่อัปเดต**:
- ไฟล์ README
- เอกสาร API
- คอมเมนต์โค้ด
- CHANGELOG

---

### /update-codemaps

**วัตถุประสงค์**: สร้าง codemaps ใหม่

**คำอธิบาย**: สร้าง code maps ใหม่สำหรับการนำทางและทำความเข้าใจ codebase

**ส่งออกไปที่**: `docs/CODEMAPS/`

---

### /ecc-guide

**วัตถุประสงค์**: คู่มือผู้ใช้ ECC

**คำอธิบาย**: แสดงคู่มือผู้ใช้ที่สมบูรณ์สำหรับ ECC plugin

---

## คำสั่งที่เกี่ยวข้อง

- `/update-docs` - อัปเดตเอกสาร
- `/update-codemaps` - อัปเดต code maps
- `/ecc-guide` - คู่มือผู้ใช้
