# คำสั่งปรับโครงสร้างและทำความสะอาด

## ภาพรวม

คำสั่งปรับโครงสร้างและทำความสะอาดใช้สำหรับการปรับโครงสร้างโค้ด การลบ dead code และการอัปเดตอัตโนมัติ

## รายการคำสั่ง

### /refactor-clean

**วัตถุประสงค์**: ลบ dead code + รวมโค้ดที่ซ้ำกัน

**คำอธิบาย**: ระบุและลบโค้ดที่ไม่ได้ใช้ รวมโค้ดที่ซ้ำกัน

**เนื้อหาการวิเคราะห์**:
- ฟังก์ชันและตัวแปรที่ไม่ได้ใช้
- บล็อกโค้ดที่ซ้ำกัน
- คอมเมนต์ที่ล้าสมัย
- import ที่ไม่ได้ใช้

**การรองรับเครื่องมือ**:
- `knip` - ตรวจจับ dead code
- `depcheck` - ตรวจจับการพึ่งพาที่ไม่ได้ใช้
- `ts-prune` - การตรวจจับ dead code ของ TypeScript

---

### /auto-update

**วัตถุประสงค์**: ความสามารถในการอัปเดตอัตโนมัติ

**คำอธิบาย**: อัปเดตการพึ่งพาและการตั้งค่าโครงการเป็นเวอร์ชันล่าสุดโดยอัตโนมัติ

**เนื้อหาที่อัปเดต**:
- การอัปเดต npm/pip packages
- การย้ายไฟล์ตั้งค่า
- การย้ายโค้ด
- การตรวจสอบความเข้ากันได้ของเวอร์ชัน

---

## คำสั่งที่เกี่ยวข้อง

- `/refactor-clean` - ปรับโครงสร้างและทำความสะอาด
- `/auto-update` - อัปเดตอัตโนมัติ
