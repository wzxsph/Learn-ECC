# 项目管理命令

## 概述

项目管理命令用于项目配置、基础设施设置和工具链管理。

## 命令列表

### /projects

**用途**: 列出已知项目 + instinct 统计

**描述**: 列出用户所有已知项目及其相关的 instinct 统计信息。

---

### /harness-audit

**用途**: 审计 agent harness 配置

**描述**: 检查和审计 Claude Code 的 harness 配置。

**检查内容**:
- 工具链配置
- 代理设置
- MCP 服务器状态
- 规则配置

---

### /model-route

**用途**: 路由任务到合适模型

**描述**: 根据任务类型选择最合适的模型（Haiku/Sonnet/Opus）。

**模型选择指南**:
- **Haiku**: 轻量级任务、代码生成、频繁调用
- **Sonnet**: 主要开发工作、复杂编码任务
- **Opus**: 架构决策、深度推理、研究分析

---

### /pm2

**用途**: PM2 进程管理器初始化

**描述**: 初始化 PM2 进程管理器配置。

---

### /setup-pm

**用途**: 配置包管理器

**描述**: 配置项目使用的包管理器。

**支持的包管理器**:
- npm
- pnpm
- yarn
- bun

**可通过** `CLAUDE_PACKAGE_MANAGER` **环境变量配置**

---

### /project-init

**用途**: 项目初始化

**描述**: 初始化新项目的标准结构和配置。

---

## 相关命令

- `/projects` - 列出项目
- `/harness-audit` - 审计配置
- `/model-route` - 模型路由
