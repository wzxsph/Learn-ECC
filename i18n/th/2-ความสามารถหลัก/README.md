# ขั้นตอนที่ 2: ความสามารถหลัก

> เชี่ยวชาญคำสั่งหลักทั้งหมด, ผูกขาดการทำงานร่วมกันของ Agent, สามารถกำหนดเอง Hooks/Skills/Rules

## ภาพรวมหลักสูตร

ขั้นตอนนี้จะพาคุณเจาะลึกการเชี่ยวชาญความสามารถหลักของ Claude Code, รวมถึงระบบคำสั่ง 75+, 6 ประเภทการทำงานร่วมกันของ Agent, Hooks กำหนดเอง, Skills พัฒนา และ Rules เขียน. เมื่อเรียนจบขั้นตอนนี้ คุณจะสามารถควบคุม Claude Code ได้อย่างสมบูรณ์, ดำเนินการ workflow พัฒนาที่กำหนดเองในระดับสูง

## เวลาเรียน

**เวลาโดยประมาณ**: 1 สัปดาห์ (2-3 ชั่วโมงต่อวัน)

## ข้อกำหนดเบื้องต้น

- เรียนจบ [ขั้นตอนที่ 1: เริ่มต้น](../1-ก้าวแรก/README.md)
- กำหนดค่าสภาพแวดล้อม Claude Code แล้ว
- คุ้นเคยการดำเนินการ terminal พื้นฐาน

## วัตถุประสงค์การเรียนรู้

เมื่อเรียนจบขั้นตอนนี้ คุณจะสามารถ:

1. **เชี่ยวชาญระบบคำสั่ง**: ผูกขาดคำสั่งทั้ง 75+ ทั้งหมด, สามารถเลือกชุดคำสั่งที่ดีที่สุดตามสถานการณ์
2. **การทำงานร่วมกันของ Agent**: เข้าใจความรับผิดชอบของ 6 ประเภท Agent, สามารถรวมกันใช้งานเพื่อเสร็จสิ้นภารกิจที่ซับซ้อน
3. **กำหนดเอง Hooks**: สามารถเขียน PreToolUse/PostToolUse/Stop Hooks
4. **พัฒนา Skills**: เรียนรู้วิธีสร้างและจัดการ Skills กำหนดเอง, สะสม pattern workflow
5. **เขียน Rules**: เขียน rules ในระดับโครงการ, กำหนดมาตรฐาน review code และคุณภาพ

## สื่อการเรียนที่มาพร้อม

หลักสูตรนี้มาพร้อมกับคู่มืออ้างอิงทางเทคนิคที่สมบูรณ์, อยู่ในไดเรกทอรีหลัก:
- [เอกสารอ้างอิง ECC](../เอกสารอ้างอิง/README.md) - มีเอกสารโดยละเอียดของคำสั่ง, Agent, Hooks, Skills, Rules ทั้งหมด

## คำแนะนำการเรียน

1. **ค่อยเป็นค่อยไป**: เรียนตามลำดับแต่ละโมดูล, เรียนจบโมดูลหนึ่งแล้วค่อยไปโมดูลถัดไป
2. **เรียนโดยการลงมือทำ**: แต่ละแบบฝึกหัดต้องลงมือทำจริง, อย่าดูอย่าทำ
3. **อ่านเอกสาร**: แต่ละโมดูลมีลิงก์ไปยังเอกสารทางเทคนิคโดยละเอียดมาพร้อม
4. **จดบันทึก**: บันทึกความรู้สึกในการใช้งานและ best practices ของตัวเอง
