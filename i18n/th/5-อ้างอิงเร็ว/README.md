# ขั้นตอนที่ 5: อ้างอิงเร็ว

คู่มือ tra cứu nhanhสำหรับคำสั่ง, Agent และ workflow ที่ใช้บ่อย

## เนื้อหา

| ตาราง tra cứu | เนื้อหา |
|--------|------|
| [Tra cứuคำสั่ง](./cheatsheet-commands.md) | คำสั่ง slash, คำสั่ง script, คำอธิบายพารามิเตอร์ |
| [Tra cứu Agent](./cheatsheet-agents.md) | รายการ Agent และความสามารถ |
| [Tra cứu workflow](./cheatsheet-workflows.md) | แม่แบบ workflow ที่ใช้บ่อย |

## คำแนะนำในการใช้

- เก็บคำสั่งที่ใช้บ่อยไว้ในบันทึก
- พิมพ์รายการ Agent เป็นเอกสารอ้างอิง
- บันทึกแม่แบบ workflow ไว้ในรายการโปรด
