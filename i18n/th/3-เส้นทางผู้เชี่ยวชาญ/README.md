# ขั้นตอนที่ 3: เส้นทางผู้เชี่ยวชาญ

ขั้นตอนนี้สำรวจคุณสมบัติขั้นสูงและการใช้งานระดับผู้เชี่ยวชาญของ ECC

## วัตถุประสงค์การเรียนรู้

เมื่อเรียนจบขั้นตอนนี้ คุณจะสามารถ:
- พัฒนาส่วนขยาย MCP (Model Context Protocol)
- สร้างระบบ Agent อัตโนมัติ
- นำ pattern การออกแบบขั้นสูงและ best practices ไปใช้

## เวลาโดยประมาณ

6-8 ชั่วโมง

## ข้อกำหนดเบื้องต้น

- เรียนจบ [ขั้นตอนที่ 2: ความสามารถหลัก](../2-ความสามารถหลัก/README.md)
- มีประสบการณ์พัฒนาโครงการที่หลากหลาย
- เข้าใจการออกแบบ architecture ระบบ
