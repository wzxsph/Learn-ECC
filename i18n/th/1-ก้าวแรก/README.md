# ขั้นตอนที่ 1: เริ่มต้น

ขั้นตอนนี้ออกแบบสำหรับผู้เริ่มต้นทั้งหมด ช่วยให้คุณเข้าใจ ECC อย่างรวดเร็วและเริ่มใช้งาน

## วัตถุประสงค์การเรียนรู้

เมื่อเรียนจบขั้นตอนนี้ คุณจะสามารถ:
- เข้าใจแนวคิดพื้นฐานของ ECC และสถานการณ์การใช้งาน
- ติดตั้งและกำหนดค่า Claude Code เสร็จสมบูรณ์
- ใช้คำสั่งที่ใช้บ่อยได้อย่างเชี่ยวชาญ
- เสร็จสิ้นภารกิจพัฒนาง่ายๆ ด้วยตัวเอง

## เวลาโดยประมาณ

2-3 ชั่วโมง

## ข้อกำหนดเบื้องต้น

- พื้นฐาน command line
- พื้นฐาน JavaScript/Node.js (เข้าใจพอ就行了)
- แนวคิดพื้นฐาน Git
