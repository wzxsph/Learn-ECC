# 第一阶段：入门

本阶段为零基础用户设计，帮助你快速了解 ECC 并上手使用。

## 学习目标

完成本阶段后，你将能够：
- 理解 ECC 的基本概念和应用场景
- 完成 Claude Code 的安装和基础配置
- 熟练使用常用命令
- 独立完成简单的开发任务

## 预计时长

2-3 小时

## 前提要求

- 命令行基础
- JavaScript/Node.js 基础（了解即可）
- Git 基础概念
