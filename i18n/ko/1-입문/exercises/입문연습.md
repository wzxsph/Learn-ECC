# 입문 연습

다음 연습을 완료하여 ECC의 기초를巩固합니다.

## 연습 1: 환경 검증

**목표**: ECC 환경이 올바르게 설치되었는지 확인

**단계**:
1. 테스트 스위트 실행: `node tests/run-all.js`
2. 모든 테스트가 통과하는지 검증
3. `.claude/` 디렉토리 구조 나열

**완료 기준**:
- 테스트가 모두 통과
- agents, skills, hooks, rules 디렉토리가 확인됨

---

## 연습 2: 슬래시 명령 사용

**목표**: 자주 사용하는 슬래시 명령에 익숙해지기

**단계**:
1. Claude Code 시작
2. 다음 명령을 순서대로 실행하고 출력을 관찰:
   - `/help` 또는 `/ecc:plan`
   - `/ecc:code-review`
3. 각 명령의 출력 형식을 기록

**완료 기준**:
- 각 명령의 기능과 출력 형식을 이해

---

## 연습 3: 단순 Agent 생성

**목표**: Agent 파일 형식 이해

**단계**:
1. `agents/` 디렉토리 아래 기존 Agent 파일 확인
2. `planner.md` 의 형식 참조
3. 단순한 `hello-agent.md` 를 생성, 기능은 인사 출력

**완료 기준**:
- 새 Agent 파일 형식이 올바름
- YAML frontmatter（name, description, tools, model）를 포함

---

## 연습 4: Hook 실행

**목표**: Hook 메커니즘 이해

**단계**:
1. `hooks/` 디렉토리 아래 Hook 설정 확인
2. `node scripts/hooks/run-with-flags.js --help` 실행（지원하는 경우）
3. Hook의 트리거 타이밍과 실행 로직 이해

**완료 기준**:
- Hook의 작동 원리를 설명할 수 있음
- PreToolUse와 PostToolUse의 차이 이해

---

## 연습 5: 종합 임무

**목표**: 작은 프로젝트 완료

**임무**: 사람 이름 포맷ユーティ리티 생성

**기능 요구사항**:
- `formatName(name)` - 「성, 이름」 형식으로 포맷
- `initials(name)` - 첫 글자省略形 반환

**실행 플로**:
1. `/ecc:plan`으로 계획
2. `/tdd`로 개발
3. `/ecc:code-review`로 리뷰

**완료 기준**:
- 2개의 함수를 완전히 구현
- 테스트 케이스를 포함
- 코드 리뷰 통과
