# 제1단계：입문

이 단계는 제로 기초 사용자를 위해 설계되어 있으며, ECC에 대한 빠른 이해와 사용 시작을 도와드립니다.

## 학습 목표

 이 단계를 완료한 후, 다음을 할 수 있습니다:
- ECC의 기본 개념과 응용 시나리오 이해
- Claude Code 설치 및 기본 설정 완료
- 자주 사용하는 명령熟练 사용
- 독립적으로 단순한 개발 임무 완료

## 예상 소요 시간

2-3시간

## 전제 조건

- 커맨드라인 기초
- JavaScript/Node.js 기초（了解程度で可）
- Git 기초 개념
