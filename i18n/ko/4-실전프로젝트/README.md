# Stage 4: 실전 프로젝트

실제 프로젝트를 통해 학습한 지식을巩固합니다.

## 학습 목표

본 단계를 완료한 후, 다음을 할 수 있습니다:
- 완전한 CLI 도구를 처음부터 구축
- 자동화 코드 검토 프로세스 구현
- 엔드투엔드 자동화 워크플로 설계
- 멀티 에이전트 협업 플랫폼 구축

## 예상 소요 시간

10-15시간

## 선수 요건

- [Stage 2: 핵심 능력](../2-핵심-능력/README.md) 완료
- 최소 하나의 스크립트 언어 마스터
- 프로젝트 개발 경험

|------|------|------|

| [프로젝트 4: Agent 시스템](./project-4-agent.md) | 멀티 Agent 협업 플랫폼 | 고급 |

하나의感兴趣的 프로젝트를深入完成하는 것을 권장합니다. 모든 프로젝트를蜻蜓点水式로 하는 것보다：
CLI 도구 → 코드 검토 → 워크플로 자동화 → Agent 시스템（난이도 순서）으로 진행하세요.
