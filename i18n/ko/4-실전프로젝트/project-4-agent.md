# 프로젝트 4: Agent 시스템

멀티 에이전트 협업 플랫폼을 구축합니다.

## 프로젝트 목표

**예상 시간**: 2-3시간

여러 전문 에이전트가 협력하여 복잡한 작업을 완료하는 멀티 에이전트 협업 시스템을 만듭니다.

## 기능 요구사항

### 1. Agent 역할
- **architect** - 시스템 설계
- **developer** - 코드 구현
- **code-reviewer** - 코드 검토
- **tester** - 테스트 엔지니어
> 참고: [코드 검토류 Agent](../참조문서/agents/1.%20代码审查类.md) | [아키텍처류 Agent](../참조문서/agents/6.%20架构类.md)

### 2. 협업 메커니즘
- 작업 배포
- 결과 집계
- 충돌 해결
- 상태 동기화

### 3. 관리 기능
- Agent 생명주기 관리
- 작업 큐
- 성능 모니터링
- 리소스 스케줄링

## 기술方案

### 메시지 프로토콜

```json
{
  "type": "task",
  "from": "architect",
  "to": "developer",
  "payload": { "task": "implement-auth", "context": {} }
}
```

### 작업 상태 머신
- `pending` - 할당 대기
- `running` - 실행 중
- `completed` - 완료
- `failed` - 실패
- `blocked` - 차단
