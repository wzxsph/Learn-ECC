## 배경 이야기

### 왜 이 프로젝트가 있나요?

영어 실력이 부족하여 ECC 영문 문서를 읽는 것이 힘들었습니다. 그래서 ECC의 명령(Commands), 에이전트(Agents), 스킬(Skills) 등의 설명을 중국어로 번역하여 이 학습 코스로 정리했습니다.

참고 문서는 이렇게诞生했습니다 —— ECC 원본 프로젝트의 내용을 번역하여 각 명령, 각 에이전트가 무엇을 하는지 설명하는 것입니다.

### 코스는 어떻게 만들어졌나요?

솔직히 말하면, 이 코스는 AI가 생성했습니다. AI는 ECC의 파일 구조를 기반으로 기본 콘텐츠를 생성했지만, 세부적인 사고와 실제 평가가 부족합니다. 내용에 문제가 있거나不够明確하면, 수정 제안을提出하세요.

## 핵심 개념

### ECC란?

**ECC** (Everything Claude Code)는 Claude Code용 플러그인 프로젝트([원본 저장소](https://github.com/affaan-m/ECC))로 다음을 제공합니다:
- **75개 명령** - 슬래시 명령（如`/plan`, `/code-review`）과 스크립트 명령
- **60개 전문 에이전트** - 코드 리뷰, 빌드 수리, 아키텍처 설계 등
- **232+ 스킬** - 재사용 가능한 워크플로 템플릿
- **Hooks 시스템** - 이벤트 기반 자동화 메커니즘
- **Rules 규칙** - 필수 개발 규범

### Learn-ECC란?

**Learn-ECC**는 ECC를 학습하기 위해 정리한 체계적 학습 경로이며, ECC 자체가 아닙니다.

「실천중학습」 설계 철학을 채택하여, 각 단계에는 이론 학습과 실천 연습이 포함됩니다. 실제 프로젝트를 완료함으로써 Claude Code의 핵심 능력을 마스터하게 됩니다.

### 참고 문서란?

**참고 문서**（./참조문서/）는 ECC 원본 프로젝트를 중국어로 번역한 것입니다. ECC의 영문 문서를 중국어로 번역하여 자신（과 당신）에게 제공합니다.

内容包括:
- 완전한 명령 설명（각 명령이 무엇을 하는지）
- 모든 에이전트의 상세한 소개
- 스킬 디렉토리 구조와 작성 규범
- Hooks 시스템의 완전한 문서
- 베스트 프랙티스와 디자인 패턴

## 이 코스를 사용하는 방법

### 권장 학습 순서

1. **단계 순서대로 학습** - 각 단계는 이전 단계基础上构建
2. **이론 + 실천 결합** - 각 장에는配套 연습이 있어, 먼저 이해하고手を動かす
3. **모든 연습 완료** - 연습은 학습 경로의 핵심 부분

### 학습 제안

- **입문 단계**：2-3시간에 완료 권장, 기본 개념과操作에 익숙해지는 것이重点
- **핵심 능력**：8-12시간 권장, 각 모듈에서 실제로手を動かす
- **전문가之路**：6-8시간 권장, 일정程度的 프로젝트 경험 필요
- **실전 프로젝트**：10-15시간 권장, 관심 있는 프로젝트를 깊이 학습

### 학습 체크포인트

각 단계를 완료한 후, 다음과 같이確認：
- 해당 단계의 모든 작업을 독립적으로 완료할 수 있다
- 핵심 개념을 다른 사람에게 설명할 수 있다
- 다른 시나리오에 응용할 수 있다

## 성공 기술

1. **점진적으로 진행**: 각 Stage를 건너뛰지 말고 학습, 각 Stage는次の基礎
2. **손을 움직여라**: 각 개념을 배웠으면 바로 실천, 보기만 하고动手하지 마라
3. **연습 완료**: 각 Stage에는配套 연습이 있어认真 완료して知識を巩固
4. **빠른 참조표 활용**: 일상 사용에서 빠른 참조표를 활용,暗記하지 마라
5. **프로젝트 중심**: Stage 4는 실제 프로젝트로 모든 지식 통합
6. **노트 작성**: 학습 감상과 문제를 기록
7. **다른 사람에게 가르치기**: 다른 사람에게 가르칠 수 있어야真正掌握

## 자주 묻는 질문 (FAQ)

### Q: 프로그래밍 기초가 필요합니까?

A: 기본적인 커맨드라인과 프로그래밍 지식이 필요합니다（JavaScript/Node.js 기초는 일부 기능을 이해하는 데 도움이 됨）하지만, 특정 언어에 숙련될 필요는 없습니다. 입문 단계는 제로 기반으로 설계되었습니다.

### Q: 특정 단계를 건너뛸 수 있습니까?

A: Stage 1-3은 순서대로 완료하는 것을 권장합니다. Stage 4는 필요에 따라 관심 있는 프로젝트를 선택할 수 있습니다. Stage 5는 언제든지 참조할 수 있습니다.

### Q: 학습 후에 무엇을 할 수 있습니까?

A: 프로덕션 레벨 Claude Code 워크플로를 마스터하고, 코드 개발, 리뷰, 테스트, 자동화를 효율적으로 수행할 수 있습니다; 커스텀 에이전트 시스템을 설계하고 구현할 수 있습니다; Hooks와 스킬을 커스터마이징할 수 있습니다.

### Q: 학습 중에 문제가 발생하면 어떻게 합니까?

A: 1) [참고 문서](./참조문서/) 관련 챕터를 참조; 2) [명령 빠른 참조](./5-빠른-참조/cheatsheet-commands.md)를 참조; 3) Issue를 제출하여 도움을 요청합니다.

### Q: 학습 효과를 어떻게 검증합니까?

A: [progress-tracker.md](./progress-tracker.md)를 사용하여 진행 상황을 추적하며, 각 단계에는 명확한 학습 목표가 있고, 모든 연습을 완료하면 해당 단계의 내용을 마스터한 것입니다.

### Q: 코스 내용이 업데이트될까요?

A: 지속적으로 업데이트될 것입니다. PR 기여를 환영합니다. 자세한 가이드는 [contributing.md](./contributing.md)를 참조하세요.
