# 코드 리뷰 담당 Agent

코드 리뷰 담당 Agent는 코드 품질, 보안 및 유지보수성의 검토를 전문으로 합니다.

## Agent 목록

| Agent 이름 | 용도 | 사용 모델 | 코어 도구 |
|------------|------|----------|----------|
| code-reviewer | 범용 코드 리뷰 | sonnet | Read, Grep, Glob, Bash |
| python-reviewer | Python 코드 리뷰 (PEP 8, 타입 힌트, 보안) | sonnet | Read, Grep, Glob, Bash |
| go-reviewer | Go 코드 리뷰 (동시성, 오류 처리, 성능) | sonnet | Read, Grep, Glob, Bash |
| kotlin-reviewer | Kotlin/Android 코드 리뷰 (코루틴, Compose) | sonnet | Read, Grep, Glob, Bash |
| rust-reviewer | Rust 코드 리뷰 (소유권, 라이프사이클 보안) | sonnet | Read, Grep, Glob, Bash |
| cpp-reviewer | C++ 코드 리뷰 (메모리 안전성, 현대 C++ 관용구) | sonnet | Read, Grep, Glob, Bash |
| flutter-reviewer | Flutter/Dart 코드 리뷰 (상태 관리, 성능) | sonnet | Read, Grep, Glob, Bash |
| typescript-reviewer | TypeScript/JavaScript 코드 리뷰 (타입 안전성) | sonnet | Read, Grep, Glob, Bash |
| swift-reviewer | Swift 코드 리뷰 (프로토콜, 동시성, ARC 메모리 관리) | sonnet | Read, Grep, Glob, Bash |
| csharp-reviewer | C# 코드 리뷰 (.NET 규칙, 비동기 패턴) | sonnet | Read, Grep, Glob, Bash |
| fsharp-reviewer | F# 코드 리뷰 (함수형 이디엄, 패턴 매칭) | sonnet | Read, Grep, Glob, Bash |
| java-reviewer | Java 코드 리뷰 (Spring Boot/Quarkus 프레임워크) | sonnet | Read, Grep, Glob, Bash |

## code-reviewer

### 이름과 용도
범용 코드 리뷰로, 코드의 품질, 보안 및 유지보수성을 주도적으로 검토합니다. 모든 코드 변경 후 필수 사용.

### 능력 설명
- 코드 품질 검사 (함수 크기, 중첩 깊이, 코드 중복)
- 보안 취약점 탐지 (SQL 인젝션, XSS, 평문 자격 증명)
- React/Next.js 패턴 검사
- Node.js/백엔드 패턴 검사
- 성능 문제 식별
- 모범 사례 제안

### 적용 시나리오
- 코드 작성 또는 수정 후
- Pull Request 리뷰
- 공유 브랜치로 병합 전

### 사용 도구 목록
- Read: 파일 내용 읽기
- Grep: 코드 패턴 검색
- Glob: 파일 찾기
- Bash: git diff 및 lint 명령 실행

### 다른 Agent와의 협력 방식
- 보안 문제 발견 시 security-reviewer로 에스컬레이션
- 빌드 오류 발견 시 build-error-resolver 사용
- 리팩토링 필요 발견 시 refactor-cleaner 사용

## python-reviewer

### 이름과 용도
Python 코드 리뷰 전문가로, PEP 8 준수, Pythonic 관용구, 타입 힌트, 보안 및 성능에 중점을 둡니다.

### 능력 설명
- PEP 8 스타일 검사
- Pythonic 관용구推广 (列表推导,枚举,with 语句)
- 타입 힌트 검증
- SQL 인젝션 감지
- 코루틴과 비동기 패턴 검사
- 프레임워크 특정 검사 (Django, FastAPI, Flask)

### 적용 시나리오
- Python 프로젝트의 모든 코드 변경
- Django/FastAPI/Flask 특정 프로젝트

### 사용 도구 목록
- Read: 파일 내용 읽기
- Grep: 코드 패턴 검색
- Glob: Python 파일 찾기
- Bash: mypy, ruff, black, bandit, pytest 실행

### 다른 Agent와의 협력 방식
- security-reviewer와 보안 검사 규칙 공유
- tdd-guide 사용해서 테스트 커버리지 확보
