# 코드 리뷰 담당 Agent

코드 리뷰 담당 Agent 专门负责코드 품질、보안和유지보수성的검토 업무。

## Agent 목록

| Agent 이름 | 용도 | 사용모델 | 코어도구 |
|------------|------|----------|----------|
| code-reviewer | 通用범용 코드 리뷰 | sonnet | Read, Grep, Glob, Bash |
| python-reviewer | Python 코드 리뷰（PEP 8、타입 힌트、보안） | sonnet | Read, Grep, Glob, Bash |
| go-reviewer | Go 코드 리뷰（동시성、오류 처리、성능） | sonnet | Read, Grep, Glob, Bash |

| cpp-reviewer | C++ 코드 리뷰（메모리 안전성、现代 C++ 惯用法） | sonnet | Read, Grep, Glob, Bash |
| flutter-reviewer | Flutter/Dart 코드 리뷰（상태 관리、성능） | sonnet | Read, Grep, Glob, Bash |

## code-reviewer

### 이름과 용도
通用범용 코드 리뷰，主动리뷰코드的품질、보안和유지보수성。모든코드变更后필수사용。

### 능력 설명
- 코드 품질 검사（함수크기、嵌套깊이、코드重复）
- 보안 취약점 탐지（SQL 인젝션、XSS、평문 자격 증명）
- React/Next.js 패턴검사
- Node.js/백엔드패턴검사
- 성능문제识别
- 모범 사례 제안

### 적용 시나리오
- 코드쓰기或修改后
- Pull Request 리뷰
- 머지到共享브랜치前

### 사용 도구 목록
- Read: 읽기파일内容
- Grep: 搜索코드패턴
- Glob: 찾기파일
- Bash: 실행 git diff 和 lint 명령

### 다른 Agent와의 협력 방식
- 发现보안문제时 Escalation 到 security-reviewer
- 发现빌드오류时사용 build-error-resolver
- 发现필요리팩토링时사용 refactor-cleaner

## python-reviewer

### 이름과 용도
Python 코드 리뷰专家，专注于 PEP 8 合规、Pythonic 惯用语、타입 힌트、보안和성능。

### 능력 설명
- PEP 8 风格검사
- Pythonic 惯用语推广（列表推导、枚举、with 语句）
- 타입 힌트검증
- SQL 인젝션检测
- 코루틴与비동기 패턴검사
- 프레임워크特定검사（Django、FastAPI、Flask）

### 적용 시나리오
- Python 프로젝트中的모든코드变更
- Django/FastAPI/Flask 特定프로젝트

### 사용 도구 목록
- Read: 읽기파일内容
- Grep: 搜索코드패턴
- Glob: 찾기 Python 파일
- Bash: 실행 mypy, ruff, black, bandit, pytest

### 다른 Agent와의 협력 방식
- 与 security-reviewer 共享보안검사규칙
- 사용 tdd-guide 确保테스트커버리지
