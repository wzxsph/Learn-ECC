# 문서와 연구명령

## 개요

문서와 연구명령用于업데이트문서、生成코드地图和研究外部资源。

## 명령列表

### /update-docs

**용도**: 업데이트프로젝트문서

**描述**: 根据코드变更자동업데이트프로젝트문서。

**업데이트内容**:
- README 파일
- API 문서
- 코드주석
- CHANGELOG

---

### /update-codemaps

**용도**: 重新生成 codemaps

**描述**: 重新生成코드地图，用于코드库导航和理解。

**输出到**: `docs/CODEMAPS/`

---

### /ecc-guide

**용도**: ECC 用户가이드

**描述**: 표시 ECC 플러그인的完整用户가이드。

---

## 相关명령

- `/update-docs` - 업데이트문서
- `/update-codemaps` - 업데이트코드地图
- `/ecc-guide` - 用户가이드
