# 프로젝트 관리명령

## 개요

프로젝트 관리명령用于프로젝트설정、인프라설정和도구链管理。

## 명령列表

### /projects

**용도**: 列出已知프로젝트 + instinct 통계

**描述**: 列出用户모든已知프로젝트及其相关的 instinct 통계정보。

---

### /harness-audit

**용도**: 审计 agent harness 설정

**描述**: 검사和审计 Claude Code 的 harness 설정。

**검사内容**:
- 도구链설정
- 프록시설정
- MCP 서버状态
- 규칙설정

---

### /model-route

**용도**: 라우팅태스크到合适모델

**描述**: 根据태스크유형选择最合适的모델（Haiku/Sonnet/Opus）。

**모델选择가이드**:
- **Haiku**: 轻量级태스크、코드生成、频繁调用
- **Sonnet**: 主要개발작업、复杂인코딩태스크
- **Opus**: 아키텍처决策、깊이추론、研究분석

---

### /pm2

**용도**: PM2 프로세스管理器초기화

**描述**: 초기화 PM2 프로세스管理器설정。

---

### /setup-pm

**용도**: 설정패키지 관리자

**描述**: 설정프로젝트사용的패키지 관리자。

**지원的패키지 관리자**:
- npm
- pnpm
- yarn
- bun

**可통과** `CLAUDE_PACKAGE_MANAGER` **环境변수설정**

---

### /project-init

**용도**: 프로젝트초기화

**描述**: 초기화新프로젝트的표준구조和설정。

---

## 相关명령

- `/projects` - 列出프로젝트
- `/harness-audit` - 审计설정
- `/model-route` - 모델라우팅
