# 프론트엔드와 디자인技能

## 개요

프론트엔드와 디자인技能用于빌드现代 Web 应用、UI/UX 설계和프론트엔드아키텍처。

## 코어技能

### frontend-patterns

**용도**: 프론트엔드개발모범 사례

**코어개념**:
- 컴포넌트아키텍처
- 상태 관리
- 성능 최적화
- 응답式설계
- 없음障碍访问

---

### frontend-design-direction

**용도**: 프론트엔드설계方向指导

**涵盖内容**:
- 설계시스템
- 설계令牌
- 主题化
- 品牌一致性

---

### frontend-slides

**용도**: 演示文稿制作

**사용场景**:
- 기술演讲
- 产品演示
- 아키텍처图绘制

---

### design-system

**용도**: 설계시스템빌드

**코어컴포넌트**:
- 色彩시스템
- 字体사양
- 컴포넌트库
- 문서站点

---

### canvas-design

**용도**: Canvas 画布설계

**사용场景**:
- 데이터可视化
- 图表绘制
- 交互式图形

---

### ui-demo

**용도**: UI 演示制作

**용도**:
- 컴포넌트演示
- 交互原型
- 视觉검증

---

## Web 기술

### web-artifacts-builder

**용도**: Web 产物빌드器

**기능**:
- HTML/CSS/JS 生成
- 原型빌드
- 静态站点生成

---

## 관련スキル

- `frontend-patterns` - 프론트엔드패턴
- `frontend-design-direction` - 프론트엔드설계方向
- `frontend-slides` - 프론트엔드幻灯片
- `design-system` - 설계시스템
- `canvas-design` - Canvas 설계
- `ui-demo` - UI 演示
