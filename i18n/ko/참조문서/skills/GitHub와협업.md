# GitHub 与协作技能

## 개요

GitHub 与协作技能用于 GitHub 操作、코드 리뷰、PR 管理和团队协作。

## 코어技能

### github-ops

**용도**: GitHub 操作与자동化

**기능**:
- Issue 管理
- PR 생성与리뷰
- 코드 리뷰자동化
- GitHub Actions 워크플로우

---

### git-workflow

**용도**: Git 워크플로우사양

**코어개념**:
- 브랜치전략
- Commit 사양
- Merge 전략
- 冲突해결

---

### repo-scan

**용도**: 코드库扫描

**체크 항목**:
- 依赖漏洞
- 코드重复
- 圈复杂度
- 문서完整性

---

## 코드 리뷰

### click-path-audit

**용도**: 点击路径审计

**사용场景**:
- 用户行为분석
- 转化率최적화
- 漏斗분석

---

## 관련スキル

- `github-ops` - GitHub 操作
- `git-workflow` - Git 워크플로우
- `repo-scan` - 코드库扫描
- `click-path-audit` - 点击路径审计
