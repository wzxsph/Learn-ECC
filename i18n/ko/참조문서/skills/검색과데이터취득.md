# 검색과 데이터 취득技能

## 개요

검색과 데이터 취득技能用于网络搜索、데이터抓取、API 查询和정보检索。

## 코어技能

### exa-search

**용도**: Exa 语义搜索

**기능**:
- 语义网页搜索
- 内容提取
- 过滤和排序
- 실시간업데이트

---

### search-first

**용도**: 搜索优先메서드论

**코어원칙**:
- 先搜索后구현
- 利用现있음해결方案
- 避免重复造轮子

---

### data-scraper-agent

**용도**: 데이터抓取프록시

**사용场景**:
- 网页데이터提取
- 구조化데이터转换
- 배치采集
- 反爬虫전략

---

### iterative-retrieval

**용도**: 迭代检索패턴

**메서드**:
- 分层检索
- 相关性反馈
- 结果최적화
- 上下文丰富

---

## MCP 서버

### mcp-server-patterns

**용도**: MCP 서버개발패턴

**코어개념**:
- 도구定义
- 资源管理
- 提示模板
- 오류 처리

---

### context7

**용도**: Context7 문서检索

**기능**:
- 库문서查询
- 코드예제가져오기
- API 引用
- 버전特定문서

---

## 관련スキル

- `exa-search` - Exa 搜索
- `search-first` - 搜索优先
- `data-scraper-agent` - 데이터抓取
- `iterative-retrieval` - 迭代检索
- `mcp-server-patterns` - MCP 서버패턴
- `context7` - Context7 문서
