# 自定义 MCP 서버개발

## 개요

MCP 서버基于 Model Context Protocol 구현，用于확장 Claude Code 的能力。本가이드介绍如何개발和통합自定义 MCP 서버。

## 서버유형

### 프로세스패턴서버

통과本地프로세스시작，适合필요本地실행或访问本地资源的서비스。

```json
{
  "command": "node",
  "args": ["/path/to/your/server.js"],
  "env": {
    "YOUR_API_KEY": "value"
  }
}
```

### HTTP 패턴서버

통과 HTTP/HTTPS 访问远程서버，适合微서비스아키텍처或云서비스。

```json
{
  "type": "http",
  "url": "https://your-mcp-server.example.com/mcp",
  "headers": {
    "Authorization": "Bearer YOUR_TOKEN"
  }
}
```

## 개발流程

### 1. 사용 MCP SDK

官方提供多种언어的 SDK：

```bash
# Node.js
npm install @modelcontextprotocol/sdk

# Python
pip install mcp
```

### 2. 구현서버

**Node.js 예제：**

```javascript
const { Server } = require('@modelcontextprotocol/sdk');
const { StdioServerTransport } = require('@modelcontextprotocol/sdk');

const server = new Server(
  {
    name: "my-custom-server",
    version: "1.0.0"
  },
  {
    capabilities: {
      tools: {},
      resources: {}
    }
  }
);

// 등록도구
server.setRequestHandler('tools/list', async () => {
  return {
    tools: [
      {
        name: "my_tool",
        description: "실행自定义操作",
        inputSchema: {
          type: "object",
          properties: {
            param: { type: "string" }
          }
        }
      }
    ]
  };
});

// 处理도구调用
server.setRequestHandler('tools/call', async (request) => {
  const { name, arguments: args } = request.params;

  if (name === "my_tool") {
    // 실행操作
    return { content: [{ type: "text", text: "结果" }] };
  }
});

// 시작서버
async function main() {
  const transport = new StdioServerTransport();
  await server.connect(transport);
}

main();
```

**Python 예제：**

```python
from mcp.server import Server
from mcp.server.stdio import stdio_server

server = Server("my-custom-server")

@server.list_tools()
async def list_tools():
    return [
        Tool(
            name="my_tool",
            description="실행自定义操作",
            inputSchema={
                "type": "object",
                "properties": {
                    "param": {"type": "string"}
                }
            }
        )
    ]

@server.call_tool()
async def call_tool(name, arguments):
    if name == "my_tool":
        # 실행操作
        return [TextContent(type="text", text="结果")]

async def main():
    async with stdio_server() as (read_stream, write_stream):
        await server.run(read_stream, write_stream, server.create_initialization_options())
```

## 설정与통합

### 1. 本地테스트

```json
{
  "mcpServers": {
    "my-custom": {
      "command": "node",
      "args": ["/absolute/path/to/server.js"],
      "env": {
        "DEBUG": "true"
      }
    }
  }
}
```

### 2. 추가到 ECC

将설정추가到 `mcp-configs/mcp-servers.json`：

```json
{
  "mcpServers": {
    "my-custom": {
      "command": "npx",
      "args": ["-y", "/path/to/your/server"],
      "env": {
        "YOUR_API_KEY": "YOUR_KEY_HERE"
      },
      "description": "你的自定义서버描述"
    }
  }
}
```

## 모범 사례

1. **오류 처리**：始终돌아가기구조化的오류응답
2. **输入검증**：검증모든输入参数，거부없음效요청
3. **타임아웃控制**：설정合理的타임아웃时间
4. **로그기록**：기록핵심操作便于디버깅
5. **资源정리**：确保연결和资源正确释放
6. **버전管理**：遵循语义化버전

## 常用도구유형

| 유형 | 용도 |
|------|------|
| `tools/list` | 列出可用도구 |
| `tools/call` | 调用도구 |
| `resources/list` | 列出资源 |
| `resources/read` | 읽기资源 |
| `prompts/list` | 列出提示模板 |
| `prompts/get` | 가져오기提示 |
