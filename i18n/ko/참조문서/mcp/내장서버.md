# 内置 MCP 서버

ECC 프로젝트在 `mcp-configs/mcp-servers.json` 中预置了多个常用 MCP 서버설정。

## 按기능분류

### 개발도구

| 서버 | 유형 | 描述 |
|--------|------|------|
| `github` | process | GitHub 操作 - PRs, issues, repos |
| `jira` | process | Jira 문제추적 - 搜索、생성、업데이트、评论、流转문제 |
| `confluence` | process | Confluence Cloud 통합 - 搜索页面、가져오기内容、探索공간 |
| `filesystem` | process | 파일시스템操作（需설정路径） |
| `memory` | process | 跨세션持久化메모리 |
| `omega-memory` | process | 增强版持久化프록시메모리，지원语义搜索、多프록시协作和知识图谱 |
| `sequential-thinking` | process | 链式思维추론 |

### 배포与인프라

| 서버 | 유형 | 描述 |
|--------|------|------|

| `cloudflare-docs` | http | Cloudflare 문서搜索 |
| `cloudflare-workers-builds` | http | Cloudflare Workers 빌드 |
| `cloudflare-workers-bindings` | http | Cloudflare Workers 绑定 |

### 데이터베이스

|--------|------|------|

| `clickhouse` | http | ClickHouse 분석查询 |

### AI 与머신러닝

|--------|------|------|
| `fal-ai` | process | fal.ai 모델的 AI 图片/视频/音频生成 |

## 快速활성화

복사필요的서버설정到 `~/.claude.json` 的 `mcpServers` 节，교체 `YOUR_*_HERE` 占位符为实际值。

**예제 - 활성화 GitHub 서버：**

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": {
        "GITHUB_PERSONAL_ACCESS_TOKEN": "ghp_your_actual_token"
      },
      "description": "GitHub operations - PRs, issues, repos"
    }
  }
}
```

## 环境변수요구

| 서버 | 必需的环境변수 |
|--------|----------------|
| `github` | `GITHUB_PERSONAL_ACCESS_TOKEN` |
| `jira` | `JIRA_URL`, `JIRA_EMAIL`, `JIRA_API_TOKEN` |
| `firecrawl` | `FIRECRAWL_API_KEY` |
| `supabase` | `--project-ref` 参数 |
| `exa-web-search` | `EXA_API_KEY` |
| `fal-ai` | `FAL_KEY` |
| `browserbase` | `BROWSERBASE_API_KEY` |
| `browser-use` | `x-browser-use-api-key` 요청头 |
| `confluence` | `CONFLUENCE_BASE_URL`, `CONFLUENCE_EMAIL`, `CONFLUENCE_API_TOKEN` |
| `evalview` | `OPENAI_API_KEY`（선택） |
