# 02-자율 에이전트

작업을 자율적으로 의사결정하고 실행할 수 있는 에이전트 시스템을 구축하는 방법을 학습합니다.

## 자율 에이전트란?

자율 에이전트（Autonomous Agent）란 以下가 가능한 지능형 시스템입니다：
- 자율적으로 작업 단계 계획
- 환경 피드백에 따라 전략 조정
- 목표 달성까지 지속적으로 실행

> 참고: [자동화 및 스크립트](../참조문서/skills/自动化与脚本.md) | [코드 검토류 Agent](../참조문서/agents/1.%20代码审查类.md)

## 학습 목표

- 자율 에이전트 아키텍처 이해
- 에이전트 의사결정 흐름 설계
- 에이전트 실행 루프 구현
- 예외 처리 및 복구

## 핵심 구성요소

### 1. 플래너（Planner）
복잡한 작업을 실행 가능한 단계로 분해합니다.

### 2. 실행기（Executor）
도구를 호출하여 구체적인 작업을 실행합니다.

### 3. 평가기（Evaluator）
작업이 완료되었는지 판단하고 실행 결과를 평가합니다.

### 4. 메모리（Memory）
대화 기록과 컨텍스트를 저장합니다.

## 에이전트 패턴

### 반응형 에이전트
현재 상태에 따라 환경 변화에 직접 응답합니다.

### 목표 지향 에이전트
명확한 목표를 중심으로 계획하고 실행합니다.

### 학습형 에이전트
경험을 통해 의사결정 전략을 개선합니다.
