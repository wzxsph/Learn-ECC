# 03-고급 패턴

ECC의 고급 디자인 패턴과 모범 사례를 마스터합니다.

## 패턴 분류

> 참고: [모범 사례](../참조문서/skills/最佳实践.md)

### 1. 워크플로 패턴
- **파이프라인 패턴** - 데이터가 여러 처리 단계를 거침
- **브로드캐스트 패턴** - 하나의 작업을 여러 실행자에게分发
- **백프레셔 패턴** - 작업 생산/소비 속도 제어

### 2. Agent 패턴
- **계층형 패턴** - 메인 Agent가 하위 Agents 조정
- **시장형 패턴** - Agents가 입찰을 통해 작업 획득
- **하이브 패턴** - 대량의同类 Agents 협력

### 3. 오류 처리 패턴
- **서킷 브레이커 패턴** - 실패 후快速 실패
- **재시도 패턴** - 지수 백오프 재시도
- **데드 레터 큐** - 처리할 수 없는 작업 일시 저장

## 성능 최적화

### 컨텍스트 관리
- 사용하지 않는 컨텍스트 즉시 정리
- 압축을 사용하여 토큰消耗 감소
- 핫/콜드 데이터 분리

### 동시성 제어
- 동시 Agent 수 제한
- 세마포어를 사용하여 리소스 제어
-优雅한 degrades 구현

## 보안

### 권한 제어
- 최소 권한 원칙
-操作 감사 로그
- 민감한操作 2차 확인

### 입력 검증
- 모든 사용자 입력 검증
- 매개변수화된 쿼리로 인젝션 방지
- 출력 내용 탈민감화
