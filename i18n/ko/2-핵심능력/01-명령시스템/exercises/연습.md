# 명령 시스템 연습

## 연습 목표

Claude Code 명령 시스템의 생성, 설정, 디버깅을 마스터한다.

## 연습 내용

### 연습 1: 기본 명령 생성
`/mycommand` 명령을 생성하여以下の機能 구현:
- 현재 시간과 프로젝트 경로 출력
- `--verbose` 옵션 지원

### 연습 2: 파라미터 명령 구현
`/search` 명령을 생성하여 이하 구현:
- 검색 키워드 파라미터 수락
- `--limit` 파라미터로 결과 수 제어
- 포맷화된 출력 표시

### 연습 3: 명령 디버깅
기존 명령을 디버깅하여 에러 처리와 로그 출력 추가.

## 템플릿

```markdown
# 내 명령

## 용도
명령의 주요 기능을 설명.

## 사용 방식
`/mycommand [옵션] <파라미터>`

## 옵션
- `--option`: 옵션 설명

## 예
`/mycommand --option value`
```

## 검증
연습 완료 후, 명령을 실행하여 기능이 정상인지 검증.
