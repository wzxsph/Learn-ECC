# Stage 2: 핵심 능력 (Core Skills Mastery)

> 모든 핵심 명령 마스터, Agent 협력掌握, Hooks/Skills/Rules 커스터마이징 가능

## 과정 개요

본 단계에서는 Claude Code의 핵심 능력을 깊이 마스터하게 됩니다. 75+ 명령 시스템, 6류 Agent 협력, Hooks 커스터마이징, Skills 개발, Rules 작성을 포함합니다. 本 단계를 완료하면 Claude Code를 완전히 제어하고高度 커스터마이징된 개발 워크플로를 실현할 수 있습니다.

## 학습 시간

**예상 시간**: 1주일（매일 2-3시간）

## 선수 요건

- [Stage 1: 입문 기초](../1-입문/README.md) 완료
- Claude Code 환경 설정 완료
- 기본 터미널 작업에 익숙

## 학습 목표

본 단계를 완료한 후, 다음을 할 수 있습니다:

1. **명령 시스템 마스터**: 모든 75+ 명령을 마스터하고, 시나리오에 따라 최적의 명령 조합을 선택할 수 있음
2. **Agent 협력**: 6류 Agent의 책임을 이해하고, 조합하여 복잡한 태스크를 완료할 수 있음
3. **Hooks 커스터마이징**: PreToolUse/PostToolUse/Stop Hooks를 작성할 수 있음
4. **Skills 개발**: 커스텀 Skills를 생성·관리하고, 워크플로 패턴을 축적할 수 있음
5. **Rules 작성**: 프로젝트 수준의 Rules를 작성하고, 코드 리뷰와 품질 기준을 커스터마이징할 수 있음
